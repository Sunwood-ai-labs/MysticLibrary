# documentation

This category groups prompts for documentation work, infographics, and slide-oriented outputs. It is centered around the `graphic-recording` family and works well as a hub before choosing a more specific branch.

  <a href="./comic/"><strong>comic</strong><br>Comic-style HTML generator prompts for visual and story-driven documentation output.</a>
  <a href="./graphic-recording/"><strong>graphic-recording</strong><br>Main hub for graphic-recording prompt families across docs, slides, and thumbnails.</a>
  <a href="./graphic-recording/docs/"><strong>graphic-recording/docs</strong><br>Editable document-style graphic-recording outputs.</a>

</div>
