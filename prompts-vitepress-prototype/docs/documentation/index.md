# documentation

ドキュメント制作・図解・スライド生成向けのプロンプトをまとめたカテゴリです。グラフィックレコーディング系が中心で、`comic` / `graphic-recording` のハブとして使えます。

<h2 class="section-heading">カテゴリ構成</h2>

<div class="category-card-grid">
  <a href="./comic/"><strong>comic</strong><br>コミック調の HTML 生成プロンプト。ストーリー表現や説明資料の視覚化に向くサブカテゴリ。</a>
  <a href="./graphic-recording/"><strong>graphic-recording</strong><br>グラレコ系のハブ。docs / slide / thumbnail の用途別ブランチへ分岐。</a>
  <a href="./graphic-recording/docs/"><strong>graphic-recording/docs</strong><br>編集可能なドキュメント形式のグラレコ出力向け。</a>
  <a href="./graphic-recording/slide/"><strong>graphic-recording/slide</strong><br>スライド生成特化のプロンプト群。</a>
  <a href="./graphic-recording/thumbnail/"><strong>graphic-recording/thumbnail</strong><br>サムネイル設計・画像生成向けプロンプト群。</a>
</div>

<h2 class="section-heading">おすすめ導線</h2>

<ol class="steps">
  <li><p><strong>まずは graphic-recording を確認</strong><br><a href="./graphic-recording/">graphic-recording</a> がこのカテゴリの中心です。</p></li>
  <li><p><strong>用途別に分岐</strong><br>資料編集なら <a href="./graphic-recording/docs/">docs</a>、発表資料なら <a href="./graphic-recording/slide/">slide</a>、サムネなら <a href="./graphic-recording/thumbnail/">thumbnail</a>。</p></li>
  <li><p><strong>トップレベル prompt を組み合わせる</strong><br><a href="./MARKWHEN_REQUIREMENTS_SPECIFICATION_V2">MARKWHEN</a> や <a href="./quarkdown_slide_prompt-v1">quarkdown</a> を要件定義・構成設計に併用。</p></li>
</ol>
