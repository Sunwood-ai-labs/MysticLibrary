## 利用時のポイント

- まずは作業目的（トリアージ / レビュー / 作成 / 運用規約）でカテゴリを選ぶ
- `intent` と `input_requirements` を確認して、入力不足を先に埋める
- 優先度判断や最終承認は人間が行う
