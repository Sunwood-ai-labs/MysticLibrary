# ディープリサーチ解説

最新技術や複雑なトピックを、知識がない人でも理解できるように解説するプロンプト集です。

## プロンプト一覧

- [ディープリサーチざっくり解説 V1](./deep-research-explainer-v1.md) - 基本的な解説プロンプト
- [ディープリサーチざっくり解説 V2 物語風](./deep-research-explainer-v2-story.md) - 高校生でも興味を持てる物語風文体
- [ディープリサーチざっくり解説 V3 ラノベ風](./deep-research-explainer-v3-light-novel.md) - ライトノベル風文体
- [ディープリサーチざっくり解説 V3.1 ラノベ風 起承転結](./deep-research-explainer-v3-1-kishoutenketsu.md) - 起承転結構成のラノベ風文体

## 使い方

1. プロンプトをコピーしてAIチャットツールに貼り付け
2. 解説したい技術やトピックを入力
3. わかりやすい解説が生成されます

## 特徴

- 専門用語を平易な言葉で説明
- 具体例を交えてわかりやすく
- 読者の前提知識を仮定しない
- 文体バリエーション（標準・物語風・ラノベ風）

## バージョン選択ガイド

| バージョン | 文体 | ターゲット | 特徴 |
|-----------|------|-----------|------|
| V1 | 標準 | 一般成人 | シンプルでわかりやすい |
| V2 | 物語風 | 高校生 | ストーリーテリング要素 |
| V3 | ラノベ風 | 若年層 | エンターテイメント性高い |
| V3.1 | ラノベ風+起承転結 | 若年層 | 構造的に完結した物語 |
