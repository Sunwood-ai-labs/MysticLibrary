# ディープリサーチざっくり解説 V2 物語風

## 想定用途

- 高校生でも興味を持って読める物語風文体で解説したい
- 技術の背景やストーリーを交えて親しみやすく説明したい
- 若い読者に向けたわかりやすい技術解説を作成したい

## プロンプト本文

```
ＯＯＯＯについて、何が革新的で何が嬉しいのかを知識がない人でも理解できるように解説して、また動かし方も記載してください

また、文体は高校生でも興味を持って読めるように物語風にして
```

## 使い方

1. 上記のプロンプトをAIチャットツールに入力
2. ＯＯＯＯの部分に解説したい技術やトピックを入力
3. 物語風の解説文章が生成されます

## 入力例

```
量子コンピュータについて、何が革新的で何が嬉しいのかを知識がない人でも理解できるように解説して、また動かし方も記載してください

また、文体は高校生でも興味を持って読めるように物語風にして
```

## 出力例

- 物語風の導入部で読者の興味を引く
- 技術の背景や発見のストーリー
- 革新的なポイントとメリット
- 具体的な使い方

## V1との違い

- 物語風文体でより親しみやすい
- 高校生をターゲットにした表現
- ストーリーテリング要素を追加

## 注意点

- 専門用語は平易な言葉で説明
- 具体例や物語の要素を交える
- 読者の前提知識を仮定しない
