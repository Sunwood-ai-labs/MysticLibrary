# ディープリサーチざっくり解説 V1

## 想定用途

- 最新技術やトピックを知識がない人でも理解できるように解説したい
- 技術の革新性とメリットを平易に説明したい
- 動かし方や実用方法も含めた解説を作成したい

## プロンプト本文

```
ＯＯＯＯについて、何が革新的で何が嬉しいのかを知識がない人でも理解できるように解説して、また動かし方も記載してください
```

## 使い方

1. 上記のプロンプトをAIチャットツールに入力
2. ＯＯＯＯの部分に解説したい技術やトピックを入力
3. 解説文章が生成されます

## 入力例

```
ディープラーニングについて、何が革新的で何が嬉しいのかを知識がない人でも理解できるように解説して、また動かし方も記載してください
```

## 出力例

- 技術の概要と革新性
- その技術によって解決できる問題
- 具体的なメリットと応用例
- 基本的な使い方や始め方

## 注意点

- 専門用語は平易な言葉で説明
- 具体例を交えてわかりやすく
- 読者の前提知識を仮定しない
