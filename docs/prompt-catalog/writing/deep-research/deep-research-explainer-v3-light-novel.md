# ディープリサーチざっくり解説 V3 ラノベ風

## 想定用途

- ライトノベル風文体で技術解説を作りたい
- ラノベファンや若年層に向けた親しみやすい解説を作りたい
- エンターテイメント性の高い技術解説を提供したい

## プロンプト本文

```
ＯＯＯＯについて、何が革新的で何が嬉しいのかを知識がない人でも理解できるように解説して、また動かし方も記載してください

また、文体は高校生でも興味を持って読めるようにラノベ風にして
```

## 使い方

1. 上記のプロンプトをAIチャットツールに入力
2. ＯＯＯＯの部分に解説したい技術やトピックを入力
3. ラノベ風の解説文章が生成されます

## 入力例

```
生成AIについて、何が革新的で何が嬉しいのかを知識がない人でも理解できるように解説して、また動かし方も記載してください

また、文体は高校生でも興味を持って読めるようにラノベ風にして
```

## 出力例

- ラノベ風の冒頭で読者を引き込む
- 主人公視点での技術体験
- 技術の「力」や「能力」としての描写
- 革新的なメリットを「魔法」のように表現
- 具体的な使用方法

## V2との違い

- ラノベ特有の表現スタイル
- よりエンターテイメント性が高い
- 「チート能力」「スキル」のようなゲーム的表現

## 注意点

- ラノベのトーンと表現を維持
- 技術的内容の正確性を保つ
- 読者の前提知識を仮定しない
