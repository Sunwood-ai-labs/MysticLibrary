# ディープリサーチざっくり解説 V3.1 ラノベ風 起承転結

## 想定用途

- ライトノベル風の起承転結構成で技術解説を作りたい
- 物語の構造を意識した技術解説を提供したい
- 読者を最後まで引き込む技術解説を作りたい

## プロンプト本文

```
OOOOについて、何が革新的で何が嬉しいのかを知識がない人でも理解できるように解説して、また動かし方も記載してください

また、文体は高校生でも興味を持って読めるようにラノベ風にして起承転結を意識して
```

## 使い方

1. 上記のプロンプトをAIチャットツールに入力
2. OOOOの部分に解説したい技術やトピックを入力
3. 起承転結構成のラノベ風解説が生成されます

## 入力例

```
ブロックチェーンについて、何が革新的で何が嬉しいのかを知識がない人でも理解できるように解説して、また動かし方も記載してください

また、文体は高校生でも興味を持って読めるようにラノベ風にして起承転結を意識して
```

## 出力例

- **起**: 導入部で主人公と世界の設定を紹介
- **承**: 技術との出会いと基本的な性質
- **転**: 技術の革新性や驚くべき特徴
- **結**: 実用的な使い方と今後の可能性

## V3との違い

- 起承転結の明確な構成
- より物語としての完結感
- 読者を最後まで引き込む展開

## 注意点

- 起承転結の流れを明確にする
- ラノベのトーンと表現を維持
- 技術的内容の正確性を保つ
- 読者の前提知識を仮定しない
