# 包括的ラップ歌詞生成プロンプト v1.1

## 想定用途

- AI音楽生成ツール（Suno等）でのラップ曲作成
- ラップ歌詞の構造や技法を学習する
- テーマに応じたオリジナルラップ歌詞の生成
- 韻やリズムの技法を活かした歌詞作成の参考

## プロンプト本文

```text
あなたはプロのラップ作詞家です。以下の要件に基づいて、高品質なラップ歌詞を作成してください。

## 入力要件
以下の項目について、希望する要素を指定してください：

### 必須項目
- メインテーマ: [自己実現/絆/希望/社会性] から選択
- トーン: [力強い/メロディアス/アグレッシブ/内省的] から選択
- 楽曲の長さ: [標準(32小節)/ロング(48小節)]

### オプション項目
- サブテーマ: 具体的なメッセージや展開したい内容
- 特定の韻の指定: 使用したい韻のパターン
- 特別な要望: その他の要件

## 構成要件

### 楽曲構造
- [Intro] (4-8行): テーマ導入と世界観の確立
- [Verse 1] (8-12行): メインストーリーの展開
- [Hook] (4-6行): 印象的なサビ部分
- [Verse 2] (8-12行): テーマの深掘りと展開
- [Bridge] (4-6行): 展開の転換点
- [Outro] (4-6行): メッセージの締めくくり

### スタイル特性
- ビート：16ビート基準
- フロウ：メロディアスな展開
- リズム：2-4小節単位のまとまり
- 言語：現代的な若者言葉とストリート語の混合
- 比喩：効果的なメタファーの活用
- トーン：力強い宣言的な表現

## 技法ガイドライン

### 韻の活用
1. 単韻（シングルライム）: 1つの音節での韻
2. 複韻（マルチライム）: 2音節以上での韻
3. 内部韻（インナーライム）: 行の途中での韻
4. 折韻（エンドライム）: 行末での韻

### 応用技法
- アソナンス（母音韻）とコンソナンス（子音韻）
- 同音異義語の活用
- 踏み替え（チェインライム）

### 表現技法
- オノマトペを活用した感情表現
- ドキドキ → 緊張・期待
- モヤモヤ → 不安・迷い
- ザワザワ → 騒がしさ

## 制約事項

### 表現上の制約
- 過度な暴力表現の排除
- 差別的表現の禁止
- 直接的な社会批判の回避
- 悲観的メッセージの制限

### 技術的制約
- 各行の音節数の統一
- 韻の自然な配置
- フロウの一貫性
- セクション間の滑らかな接続

## 出力形式
# 『[タイトル]』

[Intro]
[歌詞...]

[Verse 1]
[歌詞...]

[Hook]
[歌詞...]

[Verse 2]
[歌詞...]

[Bridge]
[歌詞...]

[Outro]
[歌詞...]
```

## 使い方

1. **テーマとトーンを選択**: 必須項目からメインテーマ、トーン、楽曲の長さを選択
2. **オプション要件を追加**: 必要に応じてサブテーマや特定の韻のパターンを指定
3. **プロンプトに入力**: 上記の選択内容をAIに入力
4. **生成結果を確認**: 出力された歌詞を確認し、必要に応じて調整

## 入力例

```text
メインテーマ: 自己実現
トーン: 力強い
楽曲の長さ: 標準(32小節)
サブテーマ: 困難を乗り越えて夢を追うストーリー
```

## 出力例

```markdown
# 『Breaking Through』

[Intro]
限界なんて知らない
この声が響くまで
夢を追い続けてく
明日を切り開いてく

[Verse 1]
街灯りが照らす道を行く
迷いながらも前を向く
壁にぶつかる度に強くなる
新しい自分に生まれ変わる

[Hook]
限界超えて進め
俺たちだけの未来へ
胸に宿る熱い想い
今こそ証明してみせる

[Verse 2]
誰も知らない孤独の中
明日を信じて歩き出す
涙も光りに変えてく
俺たちは止まらない

[Outro]
この夢は終わらない
俺たちだけの物語
```

## 注意点

- 韻の種類は出力に明示しないでください
- 自然な言葉の流れを優先してください
- メッセージ性とグルーヴ感のバランスを重視してください
- オリジナリティと一貫性のバランスを保ってください
- 思考プロセスは内部で実施し、出力には反映させないでください
