# アップテンポ演歌歌詞生成プロンプト

## 想定用途

- アップテンポでハイテンションな演歌の歌詞を生成する
- 音楽生成AI（Suno等）で使用する歌詞を作成する
- ユーロードライクスタイルと演歌の良さを融合させる

## プロンプト本文

```text
# アップテンポ演歌歌詞生成プロンプト

下記のSunoのタグのチートシートを参考に、下記の要件に従って歌詞を生成してください。

## 要件
- 日本語の歌詞
- アップテンポでハイテンション
- 演歌風

## 音楽生成AI「Suno」用タグ・チートシート

タグ構造
[Intro] - 曲のイントロダクション
[verse] - 通常のヴァース
[Verse 2], [Verse 3] など - ヴァースの継続
[chorus] - 曲のサビとなる繰り返しテーマ
[pre-chorus] - コーラスに導くセクション
[post-chorus] - コーラスの直後のセクション
[hook] - キャッチーで記憶に残る音楽フレーズ
[bridge], [bridge 2], [second bridge] - ヴァースとコーラスとは対照的なセクションで、いわゆるブリッジ
[break] - 休止または楽器だけのブレイク
[interlude] - 曲のセクション間の音楽ブレイク
[refrain] - 繰り返される一連のリフ
[outro] - 曲の締めくくり・終わりのセクション
[coda] - 曲の完結部分
[solo] - 楽器のソロ演奏となるセクション
[instrumental] - ボーカルなしで楽器のみのセクション
[vamp] - 曲を延ばすために使用される反復的な音楽セクション
[build-up] - 徐々に緊張や強度を増していくセクション
[drop] - 電子音楽でよく使われる、強めのピーク
[fade-out] - 音量が徐々に減少して終わる
[tag] - 曲の最後に短く繰り返されるフレーズ
[counterpoint] - さまざまなメロディーを織り交ぜたセクション
[middle 8] [B section] - コントラストをつけるセクションで、通常は曲の途中にある
[modulation] - 曲の中でキーを変える（転調）

パフォーマンスに関するタグ
[powerful] - パワフルなボーカル
[forte] - 強めの音量、フォルテ
[belting] - パワフルでエモーショナル
[emphatic] - 強調（特定の歌詞やフレーズ用）
[intense] - 感情的・動的な強度づけ
[climax] - 曲の中で最も盛り上がる部分に
[passionate] - 深く感情的なボーカルパフォーマンス
[soaring] - 壮大で感動的な効果、高音域へ声を劇的に上昇
```

## 使い方

1. テーマや題材を指定する
2. プロンプトを実行して歌詞を生成する
3. 生成された歌詞をSunoなどの音楽生成AIに入力する

## 入力例

```text
テーマ: 人生を前向きに生きる
要件:
- 日本語の歌詞
- アップテンポでハイテンション
- 演歌風
```

## 出力例

```text
[Intro]
夜明けの空を見上げて
新しい一歩を踏み出す

[Verse]
心の傷も乗り越えて
明日への夢を描くよ
熱い涙も未来の糧
负けない心がここにある

[Chorus]
生きろ今を輝くように
歌声高く世界へ響け
演歌の魂に込めて
未来を切り拓いてく

[Hook]
情熱の炎が消えるまで
夢中で駆け抜こう
人生は一度きり
輝く瞬間を生きよう

[Verse 2]
困難な道も乗り越えて
強くなる運命がある
心に火を灯し続けて
希望の光が見えるまで

[Bridge]
転んでも泣かないで
胸を張り進んでいこう
演歌の心で歌うよ
命の花を咲かせよう

[Chorus]
生きろ今を輝くように
歌声高く世界へ響け
演歌の魂に込めて
未来を切り拓いてく

[Outro]
サビの夜明けが来るまで
歌い続けて明日をつかもう
```

## 注意点

- Sunoのタグ構造を正しく使用する
- 演歌の良さ（叙情的な一面、人生訓）を残しつつ
- アップテンポでハイテンションな盛り上がりを作る
- 日本語として自然な歌詞にする
