# アップテンポ演歌歌詞生成プロンプト

## 想定用途

- AI音楽生成ツール（Suno等）での演歌曲作成
- アップテンポでハイテンションな演歌歌詞の生成
- 演歌スタイルの歌詞構造を学習する
- モダンな演歌アレンジの参考

## プロンプト本文

```text
あなたは演歌作詞の専門家です。以下の要件に基づいて、アップテンポでハイテンションな演歌風歌詞を作成してください。

## 要件
- 日本語の歌詞
- アップテンポでハイテンション
- 演歌風

## 楽曲構成ガイド

### 基本セクション構成
[Intro] - 曲のイントロダクション
[Verse] - 通常のヴァース
[Verse 2], [Verse 3] - ヴァースの継続
[Chorus] - 曲のサビとなる繰り返しテーマ
[Pre-Chorus] - コーラスに導くセクション
[Hook] - キャッチーで記憶に残る音楽フレーズ
[Bridge] - ヴァースとコーラスとは対照的なセクション
[Break] - 休止または楽器だけのブレイク
[Interlude] - 曲のセクション間の音楽ブレイク
[Instrumental] - ボーカルなしで楽器のみのセクション
[Build-up] - 徐々に緊張や強度を増していくセクション
[Outro] - 曲の締めくくり・終わりのセクション
[End] - 曲の完了

### パフォーマンスに関するタグ
[Powerful] - パワフルなボーカル
[Forte] - 強めの音量、フォルテ
[Belting] - パワフルでエモーショナル
[Emphatic] - 強調（特定の歌詞やフレーズ用）
[Intense] - 感情的・動的な強度づけ
[Climax] - 曲の中で最も盛り上がる部分に
[Passionate] - 深く感情的なボーカルパフォーマンス
[Soaring] - 壮大で感動的な効果、高音域へ声を劇的に上昇

## 出力形式

以下のフォーマットで歌詞を作成してください：

# 『[曲名]』

[Intro]
[イントロ歌詞...]

[Verse]
[1番の歌詞...]

[Verse 2]
[2番の歌詞...]

[Chorus]
[サビの歌詞...]

[Bridge]
[ブリッジの歌詞...]

[Chorus]
[サビの歌詞（リピート）...]

[Outro]
[アウトロ歌詞...]

[End]
```

## 使い方

1. **テーマを決定**: 作成したい歌詞のテーマやイメージを決める
2. **プロンプトに入力**: 上記のプロンプトにテーマを追加してAIに入力
3. **生成結果を確認**: 出力された歌詞を確認し、必要に応じて調整
4. **Suno等で生成**: 生成された歌詞を音楽生成AIに入力

## 入力例

```text
テーマ: 夜の東京で踊る明るい演歌
要件:
- 日本語の歌詞
- アップテンポでハイテンション
- 演歌風
```

## 注意点

- 演歌の伝統的な要素（三味線、尺八などのイメージ）を意識してください
- アップテンポに合わせて歌詞のリズムを軽快にしてください
- ハイテンションな雰囲気を維持してください
- Sunoのセクションタグを正しく使用してください
- 日本語として自然な歌詞にしてください
