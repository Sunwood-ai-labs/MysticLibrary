# 1人目がド派手なエフェクトと共に「大三元」ロン!

暗い雀荘の卓上。プレイヤーが牌を切ると同時にリズムに合わせて照明が点滅。

## プロンプト本文
~~~text
暗い雀荘の卓上。プレイヤーが牌を切ると同時にリズムに合わせて照明が点滅。 
捨てられた牌がスローモーションで回転しながら落ち、着地した瞬間にビートが爆発。 

カメラ切り替え: 
1人目がド派手なエフェクトと共に「大三元」ロン! 
背後に巨大な三つのドラゴンが浮かび上がり、音楽のベースラインにシンクロして咆哮。 

2人目は静寂から一気に光が差し込み「四暗刻単騎」ロン! 
影から浮かび上がる4組の暗刻がビートに合わせて点滅し、最後の単騎がズームイン。 

3人目は深紅のオーラをまとい「国士無双」ロン! 
13の牌がリズムに合わせて宙に舞い、最後に一閃して爆発的に集結。 

音楽はEDM調で疾走感のあるビート。映像は高速カットとスローモーションを交互に織り交ぜたMV風。 
全員の「ロン!」が重なり、画面が白くフラッシュして締める。
~~~
