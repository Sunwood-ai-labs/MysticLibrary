# Veo3 ASMR ビデオ生成プロンプト

## 想定用途

- AIビデオ生成ツール（Veo3等）でのASMRビデオ作成
- 和風の静謐な雰囲気を持つリラックスコンテンツの制作
- ミニチュア着物フィギュアを活用した美しいビジュアルコンテンツの生成
- 伝統文化と現代技術の融合を表現するアートコンテンツの制作

## プロンプト本文

このプロンプトは3つのシーンパターンに対応しています。生成したいシーンを選択してください。

## 使い方

1. **シーンを選択**: 3つのパターンから生成したいシーンを選択
2. **パラメータ調整**: 必要に応じてYAML内のパラメータを調整
3. **ビデオ生成AIに入力**: プロンプトをVeo3などのビデオ生成AIに入力
4. **生成結果を確認**: 出力されたビデオを確認

## 入力例

```text
シーン: 書道ASMR
調整: 特になし（デフォルト設定を使用）
```

## 出力例

各シーンは以下の要素を含んだビデオとして生成されます：

**書道ASMR**:
- 約15cmの着物フィギュアが半紙に文字を書く
- 筆先が紙に触れる繊細な音
- 静寂で神聖な雰囲気

**折り紙ASMR**:
- 絶世の美女フィギュアが折り紙を折る
- 和紙を折るパリパリという心地よい音
- 手作業の温かみを感じる空間

**タップダンスASMR**:
- 着物の女性がタップダンスを踊る
- タップシューズと木の床の響きのある音
- 和モダンな優雅な空間

## 注意点

- 各シーンは15-25秒の短いビデオとして設計されています
- マクロレンズ風の近接撮影を想定しています
- ASMR効果を最大化するため、繊細な音の描写に重点を置いています
- 日本の美意識（和の心）を表現するため、静寂と優雅さを重視しています
