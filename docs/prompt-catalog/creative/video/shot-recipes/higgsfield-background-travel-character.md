# シーン設定: 砂漠を旅するキャラバン。砂丘やオアシス、夜空の星座などが連続的に描かれ、物語が進む。

Style: 砂絵パフォーマンス。暗い背景にスポットライトを当てた砂のテーブルで、砂の絵が指の動きで変化する。

## プロンプト本文
~~~text
Style: 砂絵パフォーマンス。暗い背景にスポットライトを当てた砂のテーブルで、砂の絵が指の動きで変化する。

シーン設定: 砂漠を旅するキャラバン。砂丘やオアシス、夜空の星座などが連続的に描かれ、物語が進む。

タイムスタンプ:
0.00–4.00: 俯瞰の広角ショット。手が砂を広げ、波打つ砂漠の遠景とキャラバンのラクダのシルエットを描く。
4.00–8.00: 砂を払ってオアシスとヤシの木を描き、ラクダたちが水を飲む場面へ。手の動きで波紋や反射を表現する。
8.00–12.00: 画面を夜の星空に変え、手が星座を点で描く。キャラバンがテントで休むシルエットを描き、最後に砂がそっと元の平らな状態に戻る。

Cinematography:
- Lens: 35mm、真上からの固定視点で全体の変化を見せる。
- Lighting: スポットライトが砂面を照らし、手や砂に柔らかな影を作る。
- Mood: 詩的で穏やか、どこかノスタルジック。

Sound: 砂をこする音、ラクダの足音のリズム、遠くの風の音。
~~~
