# 東京AI祭ハッカソン用のデモ動画を作ってみた2⃣

&#91;Subject&#93; 主題: 東京AI祭ハッカソンの進化を予告編風に伝えるCM映像。夜の都会を疾走するカメラ映像に、ホログラムや機能紹介が次々と重なる。

## プロンプト本文
~~~text
[Subject] 主題: 東京AI祭ハッカソンの進化を予告編風に伝えるCM映像。夜の都会を疾走するカメラ映像に、ホログラムや機能紹介が次々と重なる。

[Setting] 環境: ネオンに輝く東京の夜景。高速道路を疾走する視点。摩天楼の光と車のテールランプが流線のように伸びる。

[Action] 動作: 疾走する背景に合わせて、未来的ホログラムが次々に浮かび上がる。
- 「マルチエージェントPRレビュー」→ 🦊🌿🎀のホログラムが走行中の夜景に重なってコードをレビュー。
- 「Zenn記事自動生成」→ リリースノートが光に変わり、記事がホログラム化して夜空に拡散。
- 「オリジナルCLI (gal)」→ コマンドが光のラインとなって疾走し、都市全体を包む。
- 「リアルタイム音声API」→ 声の波紋が車道に広がり、コードやIssueに変換。
クライマックスで「🚀 ワークフロー完全自動化」の文字が疾走する視点に重なり、未来的シーケンスが一気に流れる。

[Camera] カメラ: 疾走するドライバー視点。クイックカットで街の光がフラッシュのように切り替わり、ホログラムUIが空中に浮かぶ。最後は疾走から一気に引いて、夜景の上空に大きなロゴが浮かび上がる。

[Style] 映像スタイル: ハリウッド映画予告編風。青紫のネオン、サイバーな未来感、疾走感のある動き。光のトレイルやスピードブラーで迫力演出。

[Audio] 音声: 重低音の映画予告ナレーション。
- 「東京AI祭、進化は止まらない」
- 「AIがレビューし、記事を書き、デプロイする」
- 「人間はただ、アイデアを語るだけ」
SE: 疾走音、ビル風の轟き、シンセのリフ、ドラムヒット。

[Timeline] 時間構成: 10秒以内。冒頭は疾走開始→機能ごとのホログラムが次々と夜景に出現→クライマックスで「完全自動化」→最後に夜景の上にロゴと「v0.7.0 最新アップデート」。

[Constraints] 制約条件: 実在人物は描写せず、全て抽象的なホログラムやサイバー演出。疾走する背景と未来感を統一。
~~~
