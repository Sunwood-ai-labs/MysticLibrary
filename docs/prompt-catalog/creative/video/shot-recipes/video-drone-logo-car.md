# 私も2100年に流れるような未来のCM作ってみた!!

&#91;Subject&#93; 主題: 未来的な日本の家族(親と子ども)、AIロボット、スマートシティの風景

## プロンプト本文
~~~text
[Subject] 主題: 未来的な日本の家族(親と子ども)、AIロボット、スマートシティの風景 
[Setting] 環境: 2100年の日本、空に浮かぶ都市、高速で走る透明なモノレール、空を飛ぶホログラム広告、自然とテクノロジーが融合した街並み 
[Action] 動作: 家族が空中道路を走る透明な自動運転ポッドカーに乗り、ロボットが笑顔で商品を渡す。最後に大きなホログラムスクリーンに商品のロゴが表示される。 
[Camera] カメラ: ドローン風の広角ショット → 家族を追うトラッキングショット → 子どもの笑顔のクローズアップ → ホログラム広告のダイナミックなズームアウト 
[Style] 映像スタイル: 未来的・シネマティック・高彩度ネオンカラー・滑らかなCG質感 
[Audio] 音声: 未来的なシンセサイザーBGM、街の環境音(浮遊する車の音、電子音)、最後にナレーション「未来は、あなたのすぐそばに」 
[Timeline] 時間構成: 
0–3秒: 都市全景と空に浮かぶモノレールを映す 
3–6秒: 家族が透明ポッドカーで走行、AIロボットが商品を渡す 
6–9秒: 子どもの笑顔と未来的な街並みのディテール 
9–10秒: 巨大ホログラムに商品のロゴとキャッチコピー表示 
[Constraints] 制約条件: 自然な重力と動き、表情はリアルでポジティブ、広告らしい明快なテンポ
~~~
