# ボケ役が「昨日、消しゴム食べたんや!」とドヤ顔で言う。

&#91;Subject&#93; 主題: 小学生風の漫才コンビ(元気なボケ役と冷静なツッコミ役)

## プロンプト本文
~~~text
[Subject] 主題: 小学生風の漫才コンビ(元気なボケ役と冷静なツッコミ役)

[Setting] 環境: 教室の前、黒板の前に立って発表している感じ。

[Action] 動作: 
- ボケ役が「昨日、消しゴム食べたんや!」とドヤ顔で言う。 
- ツッコミ役が「食べたらアカンやろ!それおやつちゃう!」と突っ込む。 
- ボケ役が「でも、おいしかったで!フルーツの味した!」と続ける。 
- ツッコミ役が「それ、フルーツのにおい消しゴムやん!」とさらに突っ込む。

[Camera] カメラ: 
- 最初は教室全体を映す。 
- ボケのときは顔にズーム。 
- ツッコミのときはカメラがちょっと揺れて面白さを強調。

[Style] 映像スタイル: コメディ調。明るい照明。効果音に「ピコーン!」や「ドーン!」が入る。

[Audio] 音声: 
- 子どもっぽい声でのやりとり。 
- 後ろでクラスのみんなが「アハハ!」と笑う。 
- 効果音「ピコーン」「ドーン」。

[Timeline] 時間構成: 
- 0–2秒: 二人が黒板の前に登場。 
- 2–5秒: ボケ役「昨日、消しゴム食べた!」。 
- 5–7秒: ツッコミ役「食べたらアカンやろ!」。 
- 7–10秒: ボケ役「フルーツ味やった!」→ツッコミ「それ、におい消しゴムや!」

[Constraints] 制約条件: 
- 子どもでも理解できる簡単な言葉だけを使う。 
- 元気でわかりやすい表情。 
- 自然な動きとテンポ。
~~~
