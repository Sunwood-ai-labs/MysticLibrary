# [Action] 動作: カメラがケーキに寄ると、内部で「物語」が展開するように映し出される。

&#91;Subject&#93; 主題: 豪華なイチゴショートケーキがテーブル中央に置かれている

## プロンプト本文
~~~text
[Subject] 主題: 豪華なイチゴショートケーキがテーブル中央に置かれている
[Setting] 環境: 暗闇から光が差し込む豪華な宴会場
[Action] 動作: カメラがケーキに寄ると、内部で「物語」が展開するように映し出される。 
・ケーキ断面の中に若いパティシエが泡立て器を振る姿が浮かび、すぐにライバルシェフがナイフを振るう。 
・続いてロマンチックに恋人と見つめ合うシーン、裏切りの黒いソース、崩れ落ちるケーキタワー、決戦の対峙、奇跡の再生、歓喜の勝利がすべて高速でフラッシュバックのように流れる。 
・最後に再び現実のショートケーキに戻り、フォークが断面に刺さる。
[Camera] カメラ: ケーキの超クローズアップから始まり、ズームインすると内部に映画的シーンが次々と投影される。ラストでズームアウトして現実のケーキへ。 
[Style] 映像スタイル: ハリウッド超大作予告編風、ダイナミックでシネマティック 
[Audio] 音声: オーケストラ音楽の高まり、効果音(ナイフの金属音、崩壊音、歓声)、最後に日本語ナレーション「愛と戦いのすべては、この一口のために」 
[Timeline] 時間構成: 0:00–0:10 (ひとつの長いカットの中で全展開を圧縮) 
[Constraints] 制約条件: ケーキのリアルな質感を維持しつつ、内部の映像演出が自然に繋がるように
~~~
