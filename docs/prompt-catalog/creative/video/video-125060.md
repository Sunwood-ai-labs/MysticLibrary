# ウユニ塩湖で水面をスキップさせてみた!

水面にはどんな重さにも耐える絶対に破れない透明な膜がある。彼女がその膜の上を左方向に軽快にスキップしていく。絶対に膜は破れない。膜は最初から最後まで見えない。…

## プロンプト本文
~~~text
水面にはどんな重さにも耐える絶対に破れない透明な膜がある。彼女がその膜の上を左方向に軽快にスキップしていく。絶対に膜は破れない。膜は最初から最後まで見えない。カメラは歩いていく彼女を追従する。
~~~
