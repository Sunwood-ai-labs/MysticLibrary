# Low-Poly Animal Grid Generator プロンプト V1 Lite

## 想定用途

- 動物アイコンセットを素早く生成したい場合
- シンプルで使いやすいプロンプトを求めている場合
- ローポリゴンスタイルの一貫性を重視する場合
- ベクターフレンドリーなクリーンなデザインが必要な場合

## プロンプト本文

```text
Nine low-poly [ANIMAL] icons in a clean 3 × 3 grid, white background.
Each icon: triangular facets, flat solid colors only, bold black outline, retro palette (teal, coral, mustard, navy, ivory, white), tiny sparkle accent.
Vector-friendly, sharp edges, square overall.
```

## 使い方

1. `[ANIMAL]` を任意の動物の種類に置き換えます（例: cat, dog, bird, fish）
2. 画像生成AIに入力します
3. 必要に応じて色のバリエーションを調整します

## 入力例

```text
Nine low-poly bird icons in a clean 3 × 3 grid, white background.
Each icon: triangular facets, flat solid colors only, bold black outline, retro palette (teal, coral, mustard, navy, ivory, white), tiny sparkle accent.
Vector-friendly, sharp edges, square overall.
```

## 出力例

- 9つの異なる鳥のアイコンが3×3グリッドに配置されたローポリゴンイラスト
- 三角形のファセットで構成されたクリーンなスタイル
- レトロパレット（ティール、コーラル、マスタード、ネイビー、アイボリー、ホワイト）
- 太い黒いアウトラインと小さなキラキラアクセント
- ベクターフレンドリーで拡大しても鮮明なエッジ

## 注意点

- これはV1の簡易版です。詳細な仕様が必要な場合は [V1（詳細版）](./low-poly-animal-grid-generator.md) を使用してください
- アウトラインは太めで、全体的にコントラストが高くなります
- 背景は常に白になります
- シンプルな1行プロンプトなので、カスタマイズが容易です
