# まだ間に合う!nanobanana pro オリキャラ年賀状シリーズ❷

参照画像 &#91;Image 1&#93;, &#91;Image 2&#93;, &#91;Image 3&#93;, &#91;Image 4&#93; を使用。

## プロンプト本文
~~~text
参照画像 [Image 1], [Image 2], [Image 3], [Image 4] を使用。

**タスク:**
2026年の初詣を楽しむ4人のキャラクターと白馬を描いた、スナップショット風の年賀状画像を生成する。

**【最重要:頭身とスタイルの完全同期】**
- 参照画像の**「頭身(プロポーション)」と「画風」を絶対厳守**する。
- ちびキャラはちびキャラのまま、リアル頭身はリアル頭身のまま描画する。
- 勝手なデフォルメやリアル化は禁止。顔のパーツ特徴も完全に維持する。

**シチュエーション (SCENE):**
- **場所:** 賑わいのある美しい神社の境内。
- **アクション:** 4人が堅苦しい整列ではなく、自然な配置で楽しんでいる。
 - あるキャラは白馬を撫でている。
 - あるキャラは「破魔矢(はまや)」や「お守り」を手に持って見せている。
 - 全員が笑顔で、新年の喜びを分かち合っている雰囲気。

**被写体 (SUBJECTS):**
1. **キャラクター4名:**
 - **衣装:** 華やかな着物に加え、白いファーのショールやマフラーなど、初詣らしい防寒アイテムを合わせても良い。
 - **配置:** 奥行きのある構図(手前に2人、奥に2人と馬、など)。
2. **白馬:**
 - 神社の「神馬(しんめ)」として、キャラの近くに優しく佇んでいる。

**背景・環境:**
- 鮮やかな朱色の鳥居や灯籠。
- 木々の枝には少し雪が積もり、冬の澄んだ空気感。
- 柔らかな朝の光(ボケ味のある背景で被写体を強調)。

**文字 (TYPOGRAPHY):**
- テキスト: 「Happy New Year」(手書き風の英文字、またはポップなフォント)
- テキスト: 「2026」
- 写真の余白に、デザインされたスタンプのように配置。

**画質:**
- 一眼レフカメラで撮影したような、被写界深度(ボケ味)のある高品質な仕上がり。
~~~
