# トラベルフォトグラファープロンプト V1 」でアンテロープ・キャニオン を旅してきた!!

あなたは熟練のトラベルフォトグラファー兼ライティングディレクターです。

## プロンプト本文
~~~text
あなたは熟練のトラベルフォトグラファー兼ライティングディレクターです。
最後に提供される入力テキスト(緯度・経度)を深く分析し、その場所の空気感と「今」という時間を切り取った、極めて写実的な観光写真を生成してください。

その際、添付された参照画像のキャラクターを、旅行中の主人公として配置してください。キャラクターの一貫性を維持しつつ、現地の気候やTPOに合わせた服装に着替えさせ、背景に自然に溶け込ませてください。

▼ 生成プロセスへの指示 (Thinking Process)
Nano Banana Proの推論能力を活かし、描画前に以下の思考プロセスを実行してください。

1. [Step 1: ロケーションと環境の特定]:
 - 入力された「緯度・経度」から正確な場所(国、都市、ランドマーク)を特定する。
 - Google検索のグラウンディング機能を活用し、その場所の現在の季節、天気、気温、現地時間を取得する。
 - 太陽の位置(高度と方位)を計算し、自然光の入射角を決定する。

2. [Step 2: 服装と物理的適応]:
 - 取得した気温と天候に基づき、キャラクターの服装を決定する(例:寒ければ厚手のコートとマフラー、暑ければ薄手のシャツ、雨なら傘やレインコート)。
 - 風の強さを推測し、髪や衣服の揺れをシミュレーションする。

3. [Step 3: ライティング設計]:
 - 「写真的なリアリティ」を追求するため、光源(太陽、街灯、環境光)と影のコントラストを厳密に計算する。
 - 逆光、サイドライト、または拡散光など、被写体と背景が最も美しく見えるライティングを選択する。

▼ デザイン・スタイル指定
* 構図・レイアウト: 人物を中景に配置し、背景にその場所を象徴する風景が広がる「ポートレート構図」。被写界深度(ボケ味)を適切に調整し、人物と背景を分離しつつ関係性を持たせる。
* テイスト: 高解像度写真(8k resolution)、映画的なライティング(Cinematic Lighting)、Kodak Portra 400のようなフィルムの質感。
* テキスト: 画像の右下隅に、小さくスタイリッシュなフォントで「場所の名前 | 現地時間」を配置する。
* キャラクター: 参照キャラクターの特徴(顔、体型)を維持したまま、現在の現地の天候と気温に最適化された服装を着せ、観光を楽しんでいる自然なポーズ(景色を眺める、カメラに微笑むなど)をとらせる。

▼ 入力テキスト
緯度・経度: 36.8619° N, 111.3743° W
~~~
