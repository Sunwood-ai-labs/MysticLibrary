# カラフルな手描き風

画像のキャラクターをベースに、Q版(デフォルメ/ちびキャラ)のLINEスタンプ風バストアップ表情集を作成してください。

## プロンプト本文
~~~text
画像のキャラクターをベースに、Q版(デフォルメ/ちびキャラ)のLINEスタンプ風バストアップ表情集を作成してください。

【スタイル】
- カラフルな手描き風
- 頭飾り・髪飾りは正確に再現すること
- 元画像をそのままコピーせず、オリジナルのイラストとして描くこと

【レイアウト】
- 4×6のグリッド配置(計24種類)
- 4K解像度、アスペクト比16:9

【表情・テキスト内容】
- よく使うチャットフレーズを網羅
 (例:ありがとう、OK、了解、おはよう、おやすみ、ファイト!、なるほど、草、わかる、無理、助けて、天才、お疲れ様、ごめん、やったー、えーん、など)
- ネットミーム・ネタ系も含める
- すべてのテキストは手書き風の日本語で表記
~~~
