# Leave it to me!" (confident pose)

Design a set of 9 square stickers arranged in a 3x3 grid, featuring the same ad…

## プロンプト本文
~~~text
Design a set of 9 square stickers arranged in a 3x3 grid, featuring the same adorable owl character. 
Each sticker shows the owl in a different expressive pose with simple colorful background, SNS sticker style, kawaii illustration, bold outlines. 
Add text bubbles with cute short phrases in a playful font. 
The owl expressions and poses should match the following phrases:
1. "Leave it to me!" (confident pose) 
2. "Cheers!" (holding a cup, joyful) 
3. "The joke's on me." (embarrassed, scratching head) 
4. "Love you!" / "Finger heart" (making finger heart, blushing) 
5. "Don't scold me!" (teary-eyed, pleading) 
6. "I'm fine." (smiling calmly, reassuring) 
7. "OK" (casual OK hand gesture, cheerful) 
8. "My stomach is growling." (holding belly, hungry expression) 
9. "Sorry!" (bowing slightly, apologetic)
~~~
