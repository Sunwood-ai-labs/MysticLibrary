# 参照画像の頭身・特徴を維持しつつ、「マンガ家のイラスト(コミックアート)」のタッチに変換する。

参照画像 &#91;Image 1&#93;〜&#91;Image 4&#93; のキャラクターIDを使用。

## プロンプト本文
~~~text
参照画像 [Image 1]〜[Image 4] のキャラクターIDを使用。

**タスク:**
日本の「少年マンガ雑誌(Shonen Manga)」の表紙を飾るような、エネルギッシュで鮮やかな年賀状画像を生成する。

**【最重要:頭身とスタイルの完全同期】**
- 参照画像の頭身・特徴を維持しつつ、**「マンガ家のイラスト(コミックアート)」**のタッチに変換する。
- **線画:** Gペンで描いたような、強弱のあるはっきりした黒い輪郭線(Inking)。
- **塗り:** コピックマーカー(Copic Markers)や、セル画調(Cel Shading)の鮮やかな塗り。

**構図 (COMPOSITION):**
- **レイアウト:** キャラクターが画面から飛び出してくるような、ダイナミックなパース(遠近法)。
- **効果:** 集中線(Speed lines)、衝撃波、キラキラ効果などの「マンガ的エフェクト」を背景に使用。

**被写体 (SUBJECTS):**
1. **キャラクター4名:**
 - **表情:** 満面の笑み、ウィンク、叫んでいるような元気な表情。
 - **ポーズ:** ジャンプしたり、拳を突き上げたりしているアクションポーズ。
2. **白馬:**
 - マンガの相棒キャラや、召喚獣のようにカッコよく描かれている。

**文字 (TYPOGRAPHY):**
- **吹き出し:** キャラクターの口元から吹き出しが出ており、中に「あけおめ!」や「Happy New Year!」のセリフが入っている。
- **タイトル:** 「2026」の文字が、マンガのタイトルロゴのように極太でデザインされている。

**品質:**
- 印刷のドット(網点)や、紙の質感がわずかに見える、リアルな「雑誌の表紙」風の仕上がり。
~~~
