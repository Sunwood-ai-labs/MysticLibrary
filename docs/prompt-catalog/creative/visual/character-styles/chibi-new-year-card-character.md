# まだ間に合う!nanobanana pro オリキャラ年賀状シリーズ❻

参照画像 &#91;Image 1&#93;〜&#91;Image 4&#93; のキャラクターIDを使用。

## プロンプト本文
~~~text
参照画像 [Image 1]〜[Image 4] のキャラクターIDを使用。

**タスク:**
ファンタジーRPGのパッケージイラストのような、冒険の始まりを感じさせる壮大な年賀状アートを生成する。

**【最重要:頭身とスタイルの完全同期】**
- 参照画像の**「頭身(プロポーション)」と「画風」を絶対厳守**する。
- ちびキャラは「SDガンダム」や「ねんどろいど」のようなデフォルメ勇者として描画。
- リアル頭身は「FF」のようなリアルな英雄として描画。

**シチュエーション (SCENE):**
- **テーマ:** 「2026年の冒険へ」。
- **場所:** 雲海を見下ろす高い崖の上、または幻想的な草原。
- **アクション:** 風に吹かれながら、地平線から昇る朝日(初日の出)を全員で見つめている。
 - 白馬は、伝説の聖獣のように輝き、キャラを乗せている、あるいは先導している。

**被写体 (SUBJECTS):**
1. **キャラクター4名:**
 - **衣装:** 「和風ファンタジー衣装」。
 着物をベースに、鎧(アーマー)、マント、魔法の杖や剣などのファンタジー要素をミックスする。
 - **ポーズ:** 武器や杖を掲げたり、地図を指差したりして「さあ、行こう!」という雰囲気。

**背景 (ENVIRONMENT):**
- 遠くに浮かぶ天空の城、または巨大な世界樹(桜や梅の巨木)。
- キラキラと舞う魔法の粒子エフェクト。
- ドラマチックな逆光とレンズフレア。

**文字 (TYPOGRAPHY):**
- タイトルロゴ: 「謹賀新年 2026」
 - 剣や盾をモチーフにした、金属的な質感の重厚なゲームロゴ風デザイン。
 - 画面下部または上部に大きく配置。

**品質:**
- コンセプトアートの巨匠が描いたような、厚塗り(Impasto)または緻密なデジタルペイント。
~~~
