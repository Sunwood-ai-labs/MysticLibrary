# Kitsune(狐)

🦊 Kitsune(狐)

## プロンプト本文
~~~text
🦊 Kitsune(狐)
Create a minimalist black-and-white typographic illustration of a Kitsune using only the letters in the word 'KITSUNE'. Each letter forms ears, tails, and eyes, entirely from the modified letters.

👺 Tengu(天狗)
Create a minimalist black-and-white typographic illustration of a Tengu using only the letters in the word 'TENGU'. The letters shape its long nose, wings, and fan.

👹 Oni(鬼)
Create a minimalist black-and-white typographic illustration of an Oni using only the letters in the word 'ONI'. The letters form horns, fierce face, and strong stance.

🌀 Rokurokubi(ろくろ首)
Create a minimalist black-and-white typographic illustration of a Rokurokubi using only the letters in the word 'ROKUROKUBI'. The letters stretch to mimic the long twisting neck and flowing kimono.
~~~
