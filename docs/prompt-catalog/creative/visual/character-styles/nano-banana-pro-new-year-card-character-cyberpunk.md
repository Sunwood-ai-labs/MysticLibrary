# まだ間に合う!nanobanana pro オリキャラ年賀状シリーズ❹

参照画像 &#91;Image 1&#93;〜&#91;Image 4&#93; のキャラクターIDを使用。

## プロンプト本文
~~~text
参照画像 [Image 1]〜[Image 4] のキャラクターIDを使用。

**タスク:**
2026年の近未来都市を舞台にした、サイバーパンクスタイルの年賀状アートを生成する。

**画風スタイル (ART STYLE):**
- **スタイル:** 高彩度なアニメ塗り、またはセミリアルな3D CGスタイル。
- **照明:** ネオンカラー(ピンク、シアン、紫)のライティング、雨に濡れた路面の反射、ブルーム効果。
- **変更点:** 参照元の画風を捨て、SF映画のポスターのようなドラマチックなタッチに変換する。ただし「誰が誰か」わかるように、顔の特徴や髪色は維持する。

**被写体 (SUBJECTS):**
1. **キャラクター4名:**
 - **衣装:** 「サイバー・キモノ (Cyber-Kimono)」。
 伝統的な着物のシルエットを残しつつ、素材は光沢のあるビニールやカーボン、発光するLEDラインが入ったテックウェア仕様。
 - **ポーズ:** クールに立ち、ホログラムの扇子やデジタルガジェットを持っている。
2. **白馬:**
 - **外見:** メカニカルなディテールを持つ「ロボット・ホース」、または青白く発光する「エネルギー体の馬」。

**背景 (ENVIRONMENT):**
- 未来の渋谷や新宿のような、高層ビルとネオン看板がひしめく夜の街。
- 伝統的な提灯(Lanterns)がデジタルホログラムで浮いている。

**文字 (TYPOGRAPHY):**
- テキスト: 「KIN-GA-SHIN-NEN」または漢字「謹賀新年」
 - グリッチノイズが入ったような、デジタルフォントやロゴデザイン。
- テキスト: 「2026」
 - HUD(ヘッドアップディスプレイ)のような表示デザイン。

**品質:**
- 4K解像度、シネマティックな構図、未来的な雰囲気。
~~~
