# まだ間に合う!nanobanana pro オリキャラ年賀状シリーズ❺

参照画像 &#91;Image 1&#93;〜&#91;Image 4&#93; のキャラクターIDを使用。

## プロンプト本文
~~~text
参照画像 [Image 1]〜[Image 4] のキャラクターIDを使用。

**タスク:**
キャラクターを「羊毛フェルト」や「粘土(クレイ)」の人形に見立てた、ミニチュア・ジオラマ風の年賀状画像を生成する。

**画風スタイル (ART STYLE):**
- **スタイル:** ストップモーション・アニメ(Stop-motion animation)、クレイアニメ、羊毛フェルトアート。
- **質感:** 柔らかい繊維の質感(Felt texture)、指紋が少し残るような粘土の質感(Clay texture)。
- **撮影技法:** マクロレンズ撮影、ティルトシフト(Tilt-shift)効果で、ミニチュア感を強調する。ボケ味を強く。
- **変更点:** キャラクターを2〜3頭身のかわいい人形スタイルにデフォルメ変換する。

**被写体 (SUBJECTS):**
1. **キャラクター4名:**
 - **外見:** 羊毛フェルトや粘土で作られた、コロンとした可愛い人形。
 - **衣装:** 布やフェルトで作られた、温かみのある着物。
2. **白馬:**
 - 木製のおもちゃ(Wooden toy)や、ふわふわのぬいぐるみ風の白馬。

**背景 (ENVIRONMENT):**
- **セット:** 手作り感のある「お正月セット」。
 - 綿で作った雪、紙で作った富士山の書き割り、折り紙の梅の花。
 - こたつや鏡餅のミニチュア小物が置かれている。
- **照明:** 暖色の柔らかいスタジオライティング。

**文字 (TYPOGRAPHY):**
- テキスト: 「謹賀新年」
 - 木のブロックや、切り文字を置いたような立体的な表現。
- テキスト: 「2026」
 - 糸で刺繍された文字、またはクッキーのような文字。

**品質:**
- 超リアルな「素材の質感」。本物の手作りジオラマを撮影したかのような仕上がり。
~~~
