## プロンプト本文
~~~text
# Nano Banana Pro Prompt: Character-Shaped Mikoshi (Statue Style)
# Structure: 構造化型 (Structured/YAML) - Shape Override Focus

INPUT_HANDLING:
 reference_image: "[添付のキャラクター画像をここに参照させる]"
 role: "Structure shape source (形状とデザインのベースとして使用)"

SUBJECT:
 core_concept: "キャラクターの形状そのものを模した巨大な神輿 (Massive Mikoshi shaped like the character)"
 structure_definition:
 - "BODY: キャラクターの全身(または上半身)が、数メートル級の巨大な「ご神体」として造形されている"
 - "BASE: その巨大なキャラクター像が、豪華な黒漆塗りの台座と担ぎ棒 (Carrying poles) の上に鎮座している"
 - "STYLE: フィギュアや着ぐるみではなく、『伝統的な日本の祭具(木彫り・金細工・漆塗り)』としての質感"

DESIGN_DETAILS:
 material_conversion:
 # キャラクターのパーツを神輿の素材に変換
 skin_area: "Smooth painted wood or ceramic finish (滑らかな木肌または陶器のような質感)"
 clothing: "Gold leaf, Silk brocade, Lacquerware (金箔、錦織、漆塗り)"
 accessories: "Intricate metal engraving (精巧な彫金細工)"
 decorations:
 - "キャラクターの周囲に垂れ下がる鈴、房 (Tassels)、注連縄 (Shimenawa)"
 - "台座の周りには提灯 (Lanterns) が配置され、キャラクター像をライトアップしている"

ENVIRONMENT:
 scene: "日本の夜祭り、熱気あふれる通り"
 action: "法被(Happi)を着た大勢の担ぎ手たちが、この巨大な『キャラクター型神輿』を担ぎ上げている"
 visual_effects: "Dynamic low-angle shot, Depth of field, Flying sparks and confetti"

CONSTRAINTS:
 - "DO NOT render a living human character. Must look like a crafted structure/statue."
 - "Preserve the character's identity and silhouette from the reference image."
 - "Ensure the connection between the character-statue and the carrying poles is structurally solid."
~~~
