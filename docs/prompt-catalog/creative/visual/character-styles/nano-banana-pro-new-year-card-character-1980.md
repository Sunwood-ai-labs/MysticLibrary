# まだ間に合う!nanobanana pro オリキャラ年賀状シリーズ❿

参照画像 &#91;Image 1&#93;〜&#91;Image 4&#93; のキャラクターIDを使用。

## プロンプト本文
~~~text
参照画像 [Image 1]〜[Image 4] のキャラクターIDを使用。

**タスク:**
1980年代の「シティポップ(City Pop)」のレコードジャケットや、当時のアニメのような、懐かしくも新しい「ニューレトロ」な年賀状画像を生成する。

**【最重要:頭身とスタイルの完全同期】**
- 参照画像の頭身・特徴を維持しつつ、**「80年代セル画アニメ風」または「エアブラシ・イラスト風」**のタッチに変換する。
- 全体にノイズや色収差(Chromatic Aberration)をわずかに加え、アナログレコードのような質感を出す。

**シチュエーション (SCENE):**
- **テーマ:** 「ハイウェイと初日の出」。
- **背景:** パステルピンクと紫のグラデーションの空、ヤシの木ではなく「松の木」のシルエット、キラキラ光る海、または都会のビル群。
- **太陽:** 巨大な「初日の出」が、シンセウェーブ(Synthwave)風のストライプ模様やグラデーションで描かれている。

**被写体 (SUBJECTS):**
1. **キャラクター4名:**
 - **衣装:** 「レトロ・モダン」。
 着物に、80年代風のアイテム(サンバイザー、大きなプラスチックのイヤリング、ハイテクスニーカー、ウォークマン)をミックス。
 - **ポーズ:** 風になびく髪、哀愁漂う視線、または爽やかな笑顔。
2. **白馬:**
 - 浜辺やハイウェイを駆け抜ける、ペガサスのようなイメージ。
 - または、コンバーチブルカー(オープンカー)の横に立っている。

**文字 (TYPOGRAPHY):**
- メイン: 「A HAPPY NEW YEAR !!」
 - 80年代風のポップな英語フォント(ネオンサイン風、または太めの筆記体)。
- 装飾: カセットテープの日付ラベルのように「2026.1.1」と入れる。

**品質:**
- 「エモい(Emotional)」雰囲気。淡く、しかし鮮やかなパステルカラーの世界。
~~~
