# まだ間に合う!nanobanana pro オリキャラ年賀状シリーズ⓱

参照画像 &#91;Image 1&#93;〜&#91;Image 4&#93; のキャラクターIDを使用。

## プロンプト本文
~~~text
参照画像 [Image 1]〜[Image 4] のキャラクターIDを使用。

**タスク:**
光を通す「ステンドグラス(Stained Glass)」のアートスタイルで、2026年の夜明けを表現した年賀状画像を生成する。

**【最重要:頭身とスタイルの完全同期】**
- 参照画像の頭身・シルエットを維持しつつ、質感のみを**「色ガラスと鉛の枠(Lead lines)」**に変換する。
- **注意:** 顔のパーツ(目や口)をガラスの分割線で分断しないこと。「顔」は一枚のガラスに焼き付けられた絵のように、美しさを保つこと。

**画風スタイル (ART STYLE):**
- **素材:** 透明感のある色ガラス(Vibrant colored glass)、黒い鉛の輪郭線。
- **ライティング:** 絵の背後から強い太陽光(Backlight)が当たり、色が輝き、手前に色のついた影が落ちている。

**被写体 (SUBJECTS):**
1. **キャラクター4名:**
 - 教会の窓に描かれる聖人のように、あるいはアール・デコ調の人物画のように、ガラスのモザイクで表現。
 - 着物の柄は、細かいガラスのパッチワークで表現され、宝石のように輝いている。
2. **白馬:**
 - 白い乳白色のガラス(Opalescent glass)で表現された、神聖な馬。

**背景 (ENVIRONMENT):**
- アーチ状の窓枠(Gothic arch)。
- 背景のガラスは、初日の出を表す「赤・オレンジ・金」のグラデーションガラス。

**文字 (TYPOGRAPHY):**
- ガラスの一部としてデザインされた「2026」の数字。
- 黒い鉛のラインで文字が形作られている。

**品質:**
- ガラスの厚み、気泡、表面のわずかな歪みまで再現された、超写実的なマクロ撮影の質感。
~~~
