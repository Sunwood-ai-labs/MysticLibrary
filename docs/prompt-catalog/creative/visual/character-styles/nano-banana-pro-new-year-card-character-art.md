# まだ間に合う!nanobanana pro オリキャラ年賀状シリーズ❼

参照画像 &#91;Image 1&#93;〜&#91;Image 4&#93; のキャラクターIDを使用。

## プロンプト本文
~~~text
参照画像 [Image 1]〜[Image 4] のキャラクターIDを使用。

**タスク:**
アール・ヌーヴォー(Art Nouveau)様式と和風を融合させた、装飾的でエレガントなポスター風の年賀状を生成する。

**【最重要:頭身とスタイルの完全同期】**
- 参照画像の**「頭身(プロポーション)」を絶対厳守**する。
- 画風は、参照元の顔の特徴を維持しつつ、**「明確な線画と淡い水彩塗り」のイラストレーションスタイル**に統一して馴染ませる。

**構図・デザイン (COMPOSITION):**
- **スタイル:** アルフォンス・ミュシャ(Alphonse Mucha)風。
- **枠(Frame):** 画面全体が、複雑で美しい「円形」や「アーチ状」の装飾枠で囲まれている。
- **装飾モチーフ:** 梅の花、椿、松の枝、リボンが絡み合うデザイン。

**被写体 (SUBJECTS):**
1. **キャラクター4名:**
 - **衣装:** ドレープ(布のひだ)が美しい、流れるようなシルエットの着物。
 - **ポーズ:** 枠の中に美しく収まるように、優雅にポーズをとっている。髪の毛が装飾の一部のように美しくなびいている。
2. **白馬:**
 - キャラクターたちの背後に、タロットカードの絵柄のように象徴的に描かれている。
 - たてがみが植物のツタのように装飾的にデザインされている。

**色彩 (COLORS):**
- パステルカラー、クリーム色、淡い金、薄紅色の優しい色使い。
- 線画はセピア色や茶色で繊細に描く。

**文字 (TYPOGRAPHY):**
- テキスト: 「HAPPY NEW YEAR 2026」
- テキスト: 「謹賀新年」
- 装飾枠の一部としてデザインされた、レトロで装飾的なフォント。

**品質:**
- 繊細な線画、版ズレのない美しい塗り。美術館に飾られるグラフィックアートのようなクオリティ。
~~~
