## Prompt Body
```text
参照画像 [Image 1]〜[Image 4] のキャラクターIDを使用。

**タスク:**
日本の伝統カードゲーム「花札(Hanafuda)」のアートスタイルを現代的にアレンジした、グラフィックアート風の年賀状を生成する。

**【最重要:頭身とスタイルの完全同期】**
- 参照画像の頭身・画風を維持しつつ、**「フラットな塗り」と「太い輪郭線」**で花札の世界観に馴染ませる。
- 影やグラデーションを極力減らした、パキッとした「版画(Woodblock Print)」のような塗り。

**デザイン要素 (GRAPHIC ELEMENTS):**
- **色彩:** 「鮮やかな赤」「漆黒」「白」「金」の4色をベースにした、コントラストの強い配色。
- **モチーフ:** 花札特有のシンボルを背景に散りばめる。
 - 満月(Full Moon)
 - 松(Pine)、梅(Plum)、桜(Cherry Blossom)
 - 赤い短冊(Red Ribbon)
 - 盃(Sake Cup)

**被写体 (SUBJECTS):**
1. **キャラクター4名:**
 - 4人が「花札の役(カード)」の一部になっているような構図。
 - 着物の柄も、花札のような大胆な植物柄にする。
2. **白馬:**
 - 花札の動物(鹿や猪)の代わりに描かれる「神聖な白馬」。
 - 横向きのシルエットを美しく見せる伝統的な日本画の描き方。

**文字 (TYPOGRAPHY):**
- 「あけましておめでとう」や「2026」の文字を、短冊の中や、和風の枠(Cartouche)の中にデザインとして組み込む。
- 家紋のような円形のロゴデザイン。

**品質:**
- ベクターアート(Vector Art)のようにシャープで、ポスターとして印刷しても映える高いデザイン性。
```
