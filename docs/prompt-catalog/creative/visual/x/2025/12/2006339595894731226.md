## Prompt Body
```text
参照画像 [Image 1]〜[Image 4] のキャラクターIDを使用。

**タスク:**
高精細な「ドット絵(Pixel Art)」と、現代的な3Dライティング効果を融合させた、「HD-2D」スタイルの年賀状画像を生成する。

**【最重要:頭身とスタイルの完全同期】**
- **ちびキャラの場合:** 王道の「16bit ドット絵キャラクター」として描画。
- **リアル頭身の場合:** リアルな比率のまま、細密な「ハイビット・ピクセルアート」として描画。
- **共通:** ドット絵になっても「誰かわかる」ように、髪型や色の特徴を捉える。

**画風スタイル (ART STYLE):**
- **スタイル:** HD-2D、アイソメトリック(Isometric / 斜め見下ろし視点)、ボクセル(Voxel)風。
- **効果:** ドット絵なのに、夕日の逆光や、水面の反射、被写界深度(手前と奥のボケ)がリアルにかかっている「リッチな画面」。

**シチュエーション (SCENE):**
- **場所:** ピクセルで描かれた、緻密な「城下町」や「神社」。
- **アクション:** RPGの街中のシーンのように、4人がそれぞれ何かをしている(屋台を見ている、馬と話している、手を振っている)。

**被写体 (SUBJECTS):**
1. **キャラクター4名:**
 - プレイヤーキャラクターのような、愛らしいドット絵アバター。
 - 着物のドット表現(着物の柄を数ピクセルで表現する職人芸)。
2. **白馬:**
 - ゲーム内のレアな乗り物やNPCのように、美しくドット打ちされた白馬。

**文字 (TYPOGRAPHY):**
- 昔のRPGの「メッセージウィンドウ」風の枠に、「あけましておめでとう! 2026」とビットマップフォントで表示。
- または、空中に浮かぶダメージ数値のように「2026」を表示。

**品質:**
- 「懐かしいけど新しい」。ドットの粒はくっきりしているが、空気感は現代的なCG。
```
