# ビジュアルコンテンツ生成

視覚的コンテンツ（画像、イラスト、カラーマップ、マルチモーダルなど）の生成に関連するプロンプトカテゴリです。

## 収録プロンプト

- [Low-Poly Animal Grid Generator](./low-poly-animal-grid-generator.md) - 9つの動物アイコンを3×3グリッドで配置したローポリゴンスタイルのイラストを生成するプロンプト（詳細版）
- [Low-Poly Animal Grid Generator Lite](./low-poly-animal-grid-generator-lite.md) - 9つの動物アイコンを3×3グリッドで配置したローポリゴンスタイルのイラストを生成するプロンプト（簡易版）
- [断面ケーキジオラマ プロンプト](./layered-cake-diorama-prompts.md) - 断面カットされた層状ケーキがミニチュアのジオラマシーンに変換される、ハイパーリアリスティックなスチルライフ画像を生成するプロンプト集
- [カラーマップ生成プロンプト v3.0](./colormap-generator-v3.0.md) - 画像から美しいカラーパレットを抽出し、LP風インタラクティブカラーマップHTMLを生成するプロンプト
- [現実×シュール乗り物 一括生成プロンプト](./surreal-vehicle-prompt-pack.md) - 現実世界に突如現れるシュールな乗り物を静止画と動画の両方で超高品質に描写するプロンプト
- [カラーマップ生成プロンプト v2.0](./colormap-generator-v2.0.md) - 画像から美しいカラーパレットを抽出し、LP風インタラクティブカラーマップHTMLを生成するプロンプト（v2.0）
- [素数×画像生成プロンプトジェネレーター](./prime-number-image-generator.md) - キーワードから素数理論を活用した独自性の高い画像生成プロンプトを作成する
