# 既存画像の外側に、印刷・断裁用の「塗り足し」領域を自然に拡張・生成する。

余白を塗り足しプロンプトはこちら👇

## プロンプト本文
~~~text
TASK:
既存画像の外側に、印刷・断裁用の「塗り足し」領域を自然に拡張・生成する。

PRESERVE FROM ORIGINAL:
- 中央のデザイン要素(テキスト、アイコン、QRコード、色相、微細な紙の質感)を一切変更せず完全に保持する。
- レイアウトの比率や、既存の要素間の距離感を維持する。

MODIFICATION (OUTPAINTING):
- 画像の上下左右四辺に対して、背景のデザインをシームレスに延長し、キャンバスサイズを拡大する。
- 元の背景色(アイボリー/オフホワイト)と、そのテクスチャを境界線なく均一に広げる。
- 拡張された領域と元の画像の接合部(つなぎ目)を完全に消失させ、一枚の連続した背景として描画する。

CONSTRAINTS:
- 新しい文字、ロゴ、図形、装飾を絶対に追加しない。
- 境界線に段差、色ムラ、影、または不自然なエッジが発生しないようにする。
- 既存のQRコードやテキストに干渉(上書きや変形)しない。
- 最終出力は、断裁時のズレを考慮した十分な安全エリアを持つ高解像度な状態に仕上げる。
~~~
