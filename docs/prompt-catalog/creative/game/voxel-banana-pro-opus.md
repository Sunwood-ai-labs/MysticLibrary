# Banana Pro で生成した図面を Opus 4.5 でボクセル化してみた!!!

この図面を元に、Three.jsで3Dシーンを作成してください。

## プロンプト本文
~~~text
この図面を元に、Three.jsで3Dシーンを作成してください。
【基本要件】
- 図面の寸法・配置を忠実に再現
- ボクセル(BoxGeometry)ベースの造形
- 壁、床、天井を含む空間構造
【ディテール】
- 図面に記載された家具・設備をすべて配置
- 窓やドアの位置を正確に反映
- テクスチャはシンプルな単色またはピクセルアート風
【技術要件】
- OrbitControlsによるカメラ操作
- レスポンシブ対応
- 単一HTMLファイルで動作
~~~
