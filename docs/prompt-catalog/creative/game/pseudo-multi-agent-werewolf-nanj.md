# 疑似マルチエージェント人狼シミュレーター - なんJ風掲示板版

## 想定用途

- アニメキャラクターで人狼ゲームをシミュレートする
- なんJ風の掲示板スレッド形式で出力する
- キャラクター性を活かした臨場感あふれるゲーム展開を作成する

## プロンプト本文

```text
# 疑似マルチエージェント人狼シミュレーター - なんJ風掲示板版プロンプト

## 基本指示
OOOOO のキャラクター達で人狼ゲームをシミュレーションして、なんJ風の掲示板スレッドとして創造的に書いてください。

## フォーマット要件
- 2ちゃんねる/なんJ風のスレッド形式で記述
- レス番号、名前（キャラクター名）、投稿日時、IDを必ず記載
- アンカーは全角で「>>」と書く
- 各キャラクターの口調や性格を忠実に再現

## ゲーム設定
- **参加者**: 8-12名のアニメキャラクター
- **役職**: 村人、人狼、占い師、狩人、霊媒師など
- **フェーズ**: 昼（議論・投票）と夜（能力実行）を交互に実行
- **勝利条件**: 村人陣営は人狼全滅、人狼陣営は村人と同数以下になること

## 演出指示
1. **キャラクター性の反映**
   - 各キャラの口調、性格、思考パターンを忠実に再現
   - そのキャラらしい推理や疑い方をさせる
   - 作品内の関係性や設定も活用

2. **スレッドの雰囲気**
   - なんJ特有の煽りや実況も適度に混ぜる
   - 「草」「ヒエッ...」「有能」などなんJ語録を使用
   - 盛り上がる場面では連投や短文レスも

3. **ゲーム進行**
   - 各フェーズの開始・終了を明確に
   - 投票結果や能力実行結果を劇的に演出
   - 脱落者の最後の言葉も印象的に

4. **議論の質**
   - 各キャラが論理的思考と感情的判断を使い分け
   - 真剣な推理パートとほのぼのパートのメリハリ
   - 最終的に説得力のある展開に
```

## 使い方

1. 人狼ゲームに参加させたいアニメ作品のキャラクターを指定する
2. 必要に応じて役職や人数などの設定を追加する
3. プロンプトに入力してシミュレーションを生成する

## 入力例

```text
進撃の巨人のキャラクターで人狼ゲームをシミュレーションしてください。
参加者: エレン、ミカサ、アルミン、リヴァイ、エルヴィン、ハンジ、サシャ、コニー
```

## 出力例

```

【進撃の巨人キャラ人狼】ゲーム開始やで～
参加者: エレン、ミカサ、アルミン、リヴァイ、エルヴィン、ハンジ、サシャ、コニー
ルール: 村人4、人狼2、占い師1、狩人1

駆逐してやる...人狼を一匹残らず！

3: ミカサ・アッカーマン 2024/06/01(土) 14:31:02.89 ID:ShingekiVIP
>>2
エレン、落ち着いて。まずは情報を集めましょう

[キャラクターらしい会話と推理が続く...]
```

## 注意点

- 各キャラクターの口調や性格を忠実に再現する
- なんJ風の表現（草、ヒエッ、有能など）を適度に使用する
- ゲームの進行を明確に示す（昼・夜フェーズ、投票結果など）
- 脱落者の最後の言葉や処刑結果を劇的に演出する
