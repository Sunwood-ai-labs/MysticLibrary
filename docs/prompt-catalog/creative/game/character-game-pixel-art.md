# . キャラクター画像プロンプト

✅. キャラクター画像プロンプト

## プロンプト本文
~~~text
✅. キャラクター画像プロンプト
pixel art of a cute small dog character, retro game sprite style, side view, fluffy ears, round eyes, tiny paws, charming and simple design, classic 16-bit era pixel art, limited color palette, clean silhouette, pure white background 

✅. 3x3ポーズシート生成プロンプト(枠なし強調版)
Generate 9 different side-view poses of the same cute retro pixel-art dog character, arranged evenly in a 3x3 composition on a pure white background. No borders, no frames, no comic panels, no boxes, no tile outlines, no divider lines, no separators, no grid, no guide marks, no cropping marks. The image should look like 9 standalone sprite poses placed on a blank white canvas, separated only by whitespace.

✅. アニメーション化プロンプト
This is a 3x3 character sprite sheet containing 9 different poses of the same retro pixel-art dog character. Make each character move with subtle motion like an idle animation in a retro pixel game. Each frame should have slightly different movement rhythm and details, such as ear twitching, tail wagging, tiny bouncing, and breathing motion, creating an overall GIF-like animated feel.
~~~
