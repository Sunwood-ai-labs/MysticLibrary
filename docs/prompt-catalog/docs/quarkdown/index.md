# Quarkdown

テキストベースのスライド作成ツール [Quarkdown](https://github.com/iamgio/quarkdown) のプロンプトです。

## プロンプト一覧

- [Quarkdownスライド作成 V1](./quarkdown-slide-v1.md) - リリースノートや技術文書からスライドを作成

## Quarkdownとは

QuarkdownはMarkdownベースのスライド作成ツールで、以下の特徴があります：

- Markdownの記法でスライドを作成可能
- reveal.jsベースの美しいスライド
- アニメーション効果
- レイアウト制御機能
- 数式や図表のサポート

## 使い方

1. プロンプトをコピーしてAIチャットツールに貼り付け
2. スライド化したいテキストコンテンツを入力
3. 生成されたQuarkdownコードを.qmdファイルとして保存
4. Quarkdownでスライドを生成
