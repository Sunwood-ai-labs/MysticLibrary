# 教育

プログラミング書籍、オンラインチュートリアル、技術ブログ、講義資料などの教育コンテンツを、視覚的に魅力的で理解しやすいマークダウンノートブックに変換するプロンプトです。

## プロンプト一覧

- [教育用コードノートブック作成 v0](./educational-notebook-v0.md) - 技術教育コンテンツをノートブック形式に変換
- [AWS認定試験ガイドテンプレート](./aws-certification-exam-guide.md) - AWS認定試験問題の詳細解説テンプレート

## 使い方

1. プロンプトをコピーしてAIチャットツールに貼り付け
2. 教材化したいコンテンツのURLまたはテキストを入力
3. 生成されたノートブックを確認・編集

## 特徴

- マークダウン形式での出力
- コードセルと解説の明確な分離
- 日本語での詳細な解説
- loguruを使ったログ出力例
- 実行結果の提示
- 視覚的な図や表の活用
- 初心者にも分かりやすい説明

## 対応技術領域

- プログラミング言語（Python, JavaScript, Java, C++など）
- データサイエンスと機械学習
- ウェブ開発とアプリケーション構築
- データベースとクラウドテクノロジー
- システム設計とアーキテクチャ
- DevOpsとMLOps
- セキュリティとネットワーキング
