# 編集可能グラレコ v3.0 涼雅

## 想定用途

- ブラウザ上で直接編集可能なグラレコを作成したい
- ライト/ダークテーマ切り替え機能を持つ資料を作りたい
- 夏の涼と和の美学を取り入れた美しいデザインを使用したい

## プロンプト本文

```text
# 編集可能グラフィックレコーディングプロンプト v3.0 ～涼雅（りょうが）～

このプロンプトは、**夏の涼と和の美学**を現代に昇華させた**日本語編集可能インフォグラフィック HTML**を自動生成するための包括的な要件定義です。

## 🎨 デザイン仕様
### 1. カラースキーム（涼雅夏和パレット + 夜涼ダークテーマ）

#### ライトテーマ「涼雅（りょうが）」- 夏の昼
- Primary: #025E73（タイトル・重要背景）
- Secondary: #038C8C（アクセント・リンク）
- Accent: #04BFAD（ハイライト・装飾）
- Neon-Gold: #F2C166（ホバー・グローエフェクト）
- BG-Dark: #F2F2F2（基本背景）
- BG-Light: #FAFAFA（セクション背景）

#### ダークテーマ「夜涼（やりょう）」- 夏の夜
- Primary-Dark: #04BFAD（蛍の光）
- Secondary-Dark: #F2C166（星の光）
- Accent-Dark: #038C8C（夜の海面）
- Neon-Dark: #66D9EF（月光）
- BG-Dark-Dark: #0A1A1F（深い夜の海）
- BG-Light-Dark: #1A2F35（夜霧）

### 2. 編集モード & テーマ切り替えボタン（必須実装）
- 編集モードボタン：contentEditableでブラウザ上編集可能
- テーマ切り替えボタン：ライト/ダークテーマをワンクリックで切替

### 3. 涼雅装飾要素
- 背景パターン：青海波文様と風紋をSVGで軽量実装
- アクセント：魚・葉・羽を控えめに配置
- グラスモーフィズム：重要データ強調時の透明感と涼やかさ

## グラレコ化したい文章
[ここにテキストを入力]
```

## 使い方

1. プロンプトをコピーしてAIに貼り付け
2. グラレコ化したいテキストを入力
3. 生成されたHTMLをブラウザで開く
4. 「編集モード」ボタンで編集可能状態に切り替え
5. 直接テキストを編集して保存（ファイル→名前を付けてページを保存）

## 入力例

```
今日の会議では、次期プロダクトの方向性について話し合いました。
1. ターゲットユーザーの再定義
2. コア機能の優先順位付け
3. リリース時期の調整
```

## 出力例

- 編集モードボタン付きのインタラクティブなグラレコHTML
- ライト/ダークテーマ切り替え機能
- 夏の美意識に基づいた美しい配色

## 注意点

- 編集モード有効時は、ブラウザ上で直接編集できます
- テーマ切り替えはワンクリックで可能です
- 保存はブラウザの「名前を付けてページを保存」機能を使用
- フォントとアイコンの読み込みにインターネット接続が必要です
