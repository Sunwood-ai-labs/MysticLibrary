# ディープリサーチざっくり解説 V2（ストーリースタイル）

## 想定用途

- 若年層に向けた解説コンテンツを作成したい場合
- 物語のような形式で技術を紹介したい場合
- 読みやすさを重視した解説記事を作成したい場合

## プロンプト本文

```text
ＯＯＯＯについて、何が革新的で何が嬉しいのかを知識がない人でも理解できるように解説して、また動かし方も記載してください

また、文体は高校生でも興味を持って読めるように物語風にして
```

## 使い方

1. `ＯＯＯＯ`の部分に解説したいトピックを入力
2. 物語風の文体での出力を要求

## 入力例

```text
Transformerについて、何が革新的で何が嬉しいのかを知識がない人でも理解できるように解説して、また動かし方も記載してください

また、文体は高校生でも興味を持って読めるように物語風にして
```

## 出力例

物語形式で以下の内容が含まれます：

- 登場人物や設定を通じた技術の紹介
- 革新性の解説
- 利点やメリット
- 使い方の説明

## 注意点

- V1と比べて、より親しみやすい文体での解説が得られます
- 技術的な正確性と読みやすさのバランスを考慮してください
