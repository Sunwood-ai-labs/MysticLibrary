# ディープリサーチざっくり解説 V3.1（ライトノベル風・起承転結）

## 想定用途

- 起承転結の構成で技術を紹介したい場合
- 物語として完結した解説コンテンツを作成したい場合
- ライトノベル風の文体で構成的な解説を行いたい場合

## プロンプト本文

```text
OOOOについて、何が革新的で何が嬉しいのかを知識がない人でも理解できるように解説して、また動かし方も記載してください

また、文体は高校生でも興味を持って読めるようにラノベ風にして起承転結を意識して
```

## 使い方

1. `OOOO`の部分に解説したいトピックを入力
2. ライトノベル風の文体で起承転結構成の出力を要求

## 入力例

```text
生成AIについて、何が革新的で何が嬉しいのかを知識がない人でも理解できるように解説して、また動かし方も記載してください

また、文体は高校生でも興味を持って読めるようにラノベ風にして起承転結を意識して
```

## 出力例

起承転結構成で以下の内容が含まれます：

- **起**: 導入・問題提起（背景や課題の設定）
- **承**: 展開（技術の説明、革新性の解説）
- **転**: 転換（意外性、新たな視点、具体的な応用）
- **結**: 結論（まとめ、使い方、今後の展望）

## 注意点

- V3に起承転結の構成を加えたバージョンです
- 物語としての完成度を高めた構成になります
- ライトノベル特有の表現技法も活用されます
