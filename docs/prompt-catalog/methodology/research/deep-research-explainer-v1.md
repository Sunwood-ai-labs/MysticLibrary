# ディープリサーチざっくり解説 V1

## 想定用途

- 専門的なトピックを一般の人に説明したい場合
- 技術や概念の概要を把握したい場合
- ブログ記事等の解説コンテンツを作成したい場合

## プロンプト本文

```text
ＯＯＯＯについて、何が革新的で何が嬉しいのかを知識がない人でも理解できるように解説して、また動かし方も記載してください
```

## 使い方

1. `ＯＯＯＯ`の部分に解説したいトピックを入力
2. 必要に応じてターゲット読者を指定

## 入力例

```text
Deep Researchについて、何が革新的で何が嬉しいのかを知識がない人でも理解できるように解説して、また動かし方も記載してください
```

## 出力例

トピックについて、以下の構成で解説が出力されます：

- 革新性の解説
- 利点やメリット
- 使い方や操作方法

## 注意点

- 平易な言葉で説明することを明示しているため、専門用語には適宜解説が付きます
- 複雑なトピックの場合、情報が簡略化される可能性があります
