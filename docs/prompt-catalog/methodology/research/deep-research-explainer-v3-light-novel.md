# ディープリサーチざっくり解説 V3（ライトノベル風）

## 想定用途

- ライトノベルの文体で技術を紹介したい場合
- 若年層にアニメ・漫画・ゲーム文化に馴染んだ形で解説したい場合
- エンターテインメント性の高い解説コンテンツを作成したい場合

## プロンプト本文

```text
ＯＯＯＯについて、何が革新的で何が嬉しいのかを知識がない人でも理解できるように解説して、また動かし方も記載してください

また、文体は高校生でも興味を持って読めるようにラノベ風にして
```

## 使い方

1. `ＯＯＯＯ`の部分に解説したいトピックを入力
2. ライトノベル風の文体での出力を要求

## 入力例

```text
量子コンピューティングについて、何が革新的で何が嬉しいのかを知識がない人でも理解できるように解説して、また動かし方も記載してください

また、文体は高校生でも興味を持って読めるようにラノベ風にして
```

## 出力例

ライトノベル風の文体で以下の内容が含まれます：

- キャラクター設定や世界観を通じた技術の紹介
- 革新性の解説
- 利点やメリット
- 使い方の説明

## 注意点

- V2（ストーリー風）よりも更にキャラクター性を強めた文体になります
- ライトノベル特有の表現（地の文、会話調、独白など）が活用されます
