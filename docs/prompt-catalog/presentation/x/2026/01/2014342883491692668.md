## Prompt Body
```text
巨大な1枚のキャンバスを舞台に、情報を“スライドで送る”のではなく、"視点そのものを動かして見せる"プレゼンにしてほしい。
全体像を俯瞰してから重要箇所へ寄ったり、関連する話題へ滑らかに移動したりと、"ズームと移動を軸にした映像的な流れ"で展開する。結果として、聴衆の注意を自然に誘導し、印象に残るストーリー体験を作る。

《狙う表現・要件》

・キャンバス基盤の構成":情報は1枚の面に配置し、章立ては“場所”で管理する。
・自由度の高い進行":決めた順路で見せてもよいし、会話の流れに合わせて寄り道・戻りもできる設計。
・カメラワーク演出":パン(視点移動)/ズーム/フォーカスを使い、必要な情報へ視線を導く。
・理解を助ける視点遷移":全体→詳細、比較対象→関連トピックなど、つながりが直感的にわかる移動を重視。
・利用シーン":企業紹介、研修・教材、展示・サイネージなど、短尺でも長尺でも成立する表現にする。
```
