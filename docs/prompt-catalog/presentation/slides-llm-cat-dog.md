## プロンプト本文
~~~text
# コンテンツの指示
「LLMの仕組み(次単語予測)」を解説するスライドを作成して。
* **見出し:** 「確率という名のコンパス」
* **ビジュアル:** ポラロイド写真の中に、「分かれ道」のスケッチが描かれている。道の先には「Cat (70%)」「Dog (20%)」「Car (10%)」といった単語が浮かんでおり、AIが確率の高い道を選ぼうとしている様子。
* **本文:** 「彼は言葉の地図を持っている。次にどの道(単語)へ進むべきか、膨大な学習データというコンパスが教えてくれるようだ。」

# デザイン・スタイルの指定(厳守)
* **レイアウト:** ポラロイド写真は少し傾けて配置。横にマスキングテープで留められたメモ書きがある。
* **タイポグラフィ:** 見出しは手書き風、解説文はタイプライター風。
* **背景:** 控えめなリネン調のテクスチャ。
* **全体的なムード:** 旅先での発見、手作り感、知的探求。
~~~
