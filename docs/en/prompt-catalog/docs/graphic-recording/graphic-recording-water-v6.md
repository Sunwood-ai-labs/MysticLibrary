# Graphic Recording Infographic V6 - Water Theme

## Intended Use

- Create flexible, adaptive information representation "like water"
- Generate infographics with flowing movement and animation effects
- Build information structures using multiple handwritten-style speech bubbles

## Prompt

```text
# Graphic Recording Infographic Transformation Prompt V6 - Water

## Purpose
Transform the following content into a Japanese graphic recording-style HTML infographic utilizing **large Font Awesome icons**, **diverse handwritten-style speech bubbles**, and **sophisticated animations with a water theme**.

## Design Specifications
### 1. Extended Color Scheme
- Water-1: #007cb7
- Water-2: #83b4b6
- Water-3: #e6eceb
- Water-4: #006793
- Water-5: #00abc5
- Water-Accent-1: #39d2e3
- Water-Accent-2: #004d71
- Water-Highlight: #ebf7f9

### 2. Sophisticated Graphic Recording Elements
- Place entire content on complex gradient background
- **One extra-large theme icon per card** (fa-5x~fa-7x)
- **Card content in iconized hierarchical list format**
- **Multiple handwritten-style speech bubble variations** (with ripple effect)
- Sections organized as cards floating on water surface

### 3. Sophisticated Typography
- Title: 36-42px, with extra-large related icon (fa-6x or larger)
- Card title: 24-28px, with large icon (fa-5x)
- Font specification:
  ```html
  <style>
  @import url('https://fonts.googleapis.com/css2?family=Yeseva+One&family=Kaisei+Decol&family=Yomogi&family=Zen+Kurenaido&family=JetBrains+Mono:wght@300;400;500&display=swap');
  @import url('https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css');
  </style>
  ```

### 4. Dynamic Layout (Advanced Water Theme)
- Overall: **Responsive grid layout** (4 columns)
- Each card: Floating-up animation from water surface
- Inter-section relationships: SVG path wavy connector lines

## Content to Transform
[Enter text here]
```

## How to Use

1. Copy the prompt and paste into AI
2. Enter text to transform into infographic
3. Save generated HTML and display in browser

## Input Example

```
Explain Bruce Lee's "Be water, my friend" philosophy.
Water is the softest substance yet can withstand the hardest.
Having no shape, it adapts to any container.
```

## Output Example

An infographic visually expressing the water characteristics of "flexibility," "adaptability," and "fluidity" will be generated.

## Notes

- Color usage and animations are adjusted for the water theme
- Speech bubbles and cards include hover effects
- Verify in browser due to complex animations
