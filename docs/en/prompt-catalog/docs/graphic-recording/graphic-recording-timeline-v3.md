# Galaxy Timeline Graphic Recording V3

## Intended Use

- Express historical events as visually appealing timelines
- Show company development history or project progress chronologically
- Represent space/astronomy-related content with appropriate theming

## Prompt

```text
# Galaxy Timeline Graphic Recording HTML Creation Prompt V3

## Purpose
Transform the following content into a timeline-style graphic recording HTML with a fantasy and pop space theme.

## Design Specifications
### 1. Color Scheme
- Illustration-1: #5BA67B
- Illustration-2: #F2BB13
- Illustration-3: #F2DB94
- Illustration-4: #BF8665
- Illustration-5: #BF190A

### 2. Space Theme Visual Effects
- Dark space background (#080808) with scattered small stars
- Large "sun" light source positioned in upper left
- Circular markers designed as colorful "planets"
- Animation giving the light source a breathing movement

### 3. Timeline Structure
- Vertical timeline in the center
- Items arranged alternately left and right
- Each item has a planet-style round marker
- Quick navigation selector at bottom or right

### 4. Typography and Text Processing
- Title: Glowing effect like stars in the night sky
- Dates: Emphasized with white background
- Event titles: Displayed in bright palette colors
- Font specification:
  ```html
  <style>
  @import url('https://fonts.googleapis.com/css2?family=Kaisei+Decol&family=Yomogi&family=Zen+Kurenaido&display=swap');
  </style>
  ```

### 5. Visual Elements and Emphasis
- Font Awesome icons to visualize each event
- Keyword highlighting (gradient underline effect)
- Translucent effects and blur for depth
- Hover animations to make items float

## Content to Transform
[Enter timeline text here]
```

## How to Use

1. Copy the prompt and paste into AI
2. Paste timeline text after "Content to Transform"
3. Save generated HTML and display in browser

## Input Example

```
January 2020 - Project launch
June 2020 - First prototype completed
March 2021 - Beta release
December 2021 - Official launch
June 2022 - 100,000 users reached
```

## Output Example

A unique and attractive timeline infographic floating in space will be generated.

## Notes

- Text is displayed in bright colors due to dark background
- Animation effects included - verify in browser
- Specify date information clearly
