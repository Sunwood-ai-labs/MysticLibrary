# Galaxy Timeline Graphic Recording V3a

## Intended Use

- Create fantastic and pop atmosphere timeline with galaxy theme
- Create vibrant timeline using Tropical Fruit Punch color palette
- Generate mysterious timeline with planet-style markers and stardust background

## Prompt

```text
# Graphic Recording (Graleco) HTML Creation Prompt with Galaxy Timeline V3

## Purpose
Please convert the following content into a timeline-style graphic recording HTML with a fantastic and pop galaxy theme. Strive for the highest level in both information design and visual design. Create a mysterious and visually appealing timeline using handwritten-style elements and galaxy backgrounds.

## Design Specifications
### 1. Color Scheme (Tropical Fruit Punch)
```
<palette>
<color name='Tropical-Fruit-Punch-1' rgb='146152' r='19' g='96' b='82' />
<color name='Tropical-Fruit-Punch-2' rgb='44803F' r='67' g='127' b='62' />
<color name='Tropical-Fruit-Punch-3' rgb='B4CF66' r='179' g='206' b='101' />
<color name='Tropical-Fruit-Punch-4' rgb='FFEC5C' r='255' g='235' b='92' />
<color name='Tropical-Fruit-Punch-5' rgb='FF5A33' r='255' g='90' b='50' />
</palette>
```

### 2. Galaxy Theme Visual Effects
- Dark galaxy background (#080808) with scattered small stars
- Large "sun-like" light source in upper left (using palette primary colors)
- Add breathing animation to light source
- Circular markers are colorful "planet-like" designs
- Apply fine stardust-like background texture throughout

### 3. Timeline Structure
- Place vertical timeline in center
- Arrange items alternately left and right
- Add planet-style round markers to each item
- Design year representation as floating cards
- Place quick navigation selector at bottom or right
- Add glowing effects to markers and selectors

### 4. Typography and Text Processing
- Title: Add glowing effect like stars in night sky
- Date: Emphasize as floating on white background
- Event title: Display in bright palette colors (Tropical-Fruit-Punch-3, 4)
- Detail text: Bright text on semi-transparent background
- Font specification: Kaisei Decol, Yomogi, Zen Kurenaido

### 5. Visual Elements and Emphasis
- Visualize each event content with Font Awesome icons
- Keyword highlights (gradient underline effect)
- Ensure appropriate spacing between items for visibility
- Create depth with semi-transparent effects and blur

---

## Content to Convert
[Enter text content to convert to timeline here]
```
