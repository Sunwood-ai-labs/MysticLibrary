# Graphic Recording Thumbnail V2 Japanese Modern

## Intended Use

- Create thumbnails with Japanese Modern (Wa-Modern) design
- Generate high-quality 16:9 thumbnails like top-tier consultants
- Fuse Japanese traditional colors with modern design

## Prompt

```text
# Graphic Recording Thumbnail Prompt V2

## Overview
This prompt generates high-quality blog article thumbnails like top-tier consultants create. Create visually polished 16:9 aspect ratio infographic thumbnails that convey content at a glance. Use the color palette format below as reference to create a modern and stylish color palette.

## Basic Structure
```
Create a thumbnail for [Article Title] in HTML/CSS
Make it 16:9 size (1920x1080px)
Use Font Awesome icons to enhance infographic readability and professionalism
Also use the color map and fonts below
```

## Color Palette Definition (Japanese Modern)
- Palette combining Japanese traditional colors with modern colors
- Examples: Indigo, Vermilion, Mouse Gray, Generation Color, Gold Tea Color, etc.

## Font Definition (Japanese fonts recommended)
- Kaisei Decol (Bold 700): For titles
- Zen Kurenaido (Regular 400): For subtitles

## Design Elements
Include the following elements:
1. **Title Section**:
   - Main title (large prominent font - use Kaisei Decol)
   - Subtitle (supplementary info - use Zen Kurenaido)
   - Relevant Font Awesome icons
2. **Visual Elements**:
   - Related graphics/illustrations on the right
   - Badges and labels (showing topics or categories)
3. **Content Features**:
   - Display 3-4 main points with icons
   - Each point should be short and concise
4. **Professional Elements**:
   - Gradient background
   - Subtle grid pattern
   - Appropriate shadow effects

---

## Content to Thumbnail
[Enter article title and content to thumbnailize here]
```

## How to Use

1. Copy the prompt and paste into AI chat tool
2. Enter the article information you want to thumbnailize in the content section
3. Display the generated HTML in browser
4. Save as image via screenshot

## Input Example

```
## Content to Thumbnail
Title: Efficient Learning Methods in the AI Era

Points:
- Accelerated understanding using AI tools
- Practical project-based learning
- Knowledge sharing in community
```

## Output Example

Professional thumbnail HTML with Japanese Modern color palette in 1920x1080px 16:9 aspect ratio will be generated.

## Notes

- Japanese Modern design fuses Japanese aesthetic with modernity
- Color palette is automatically adjusted
- Screenshot required when using as thumbnail
