# Education

Prompts for transforming educational content such as programming books, online tutorials, technical blogs, and lecture materials into visually appealing and easy-to-understand markdown notebooks.

## Prompt List

- [Educational Code Notebook Creation v0](./educational-notebook-v0.md) - Transform technical educational content into notebook format
- [AWS Certification Exam Guide Template](./aws-certification-exam-guide.md) - Template for detailed AWS certification exam question explanations

## How to Use

1. Copy the prompt and paste into an AI chat tool
2. Input the URL or text of the content you want to transform
3. Review and edit the generated notebook

## Features

- Markdown format output
- Clear separation of code cells and explanations
- Detailed explanations in Japanese
- Loguru logging examples
- Execution results presentation
- Visual figures and tables
- Beginner-friendly explanations

- Databases and cloud technologies
- System design and architecture
- DevOps and MLOps
