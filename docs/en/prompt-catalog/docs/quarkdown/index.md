# Quarkdown

Prompts for [Quarkdown](https://github.com/iamgio/quarkdown), a text-based slide creation tool.

## Prompts List

- [Quarkdown Slide Creation V1](./quarkdown-slide-v1.md) - Create slides from release notes or technical documentation

## What is Quarkdown

Quarkdown is a Markdown-based slide creation tool with the following features:

- Create slides using Markdown syntax
- Beautiful reveal.js-based slides
- Animation effects
- Layout control features
- Support for formulas and diagrams

## How to Use

1. Copy the prompt and paste it into your AI chat tool
2. Input the text content you want to convert to slides
3. Save the generated Quarkdown code as a .qmd file
4. Generate slides with Quarkdown
