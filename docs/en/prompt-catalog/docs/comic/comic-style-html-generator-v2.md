# Comic Style HTML Generator v2.0

Prompt for converting text and articles to 4-panel comic style HTML format.

## Overview

Transforms technical documents, news, and explanatory articles into visual, accessible comic-style HTML.

## Key Features

- **Character Setup**: Uses consistent characters (2-4)
- **Story Structure**: Introduction → Problem → Solution → Details → Summary
- **HTML Structure**: Complete HTML with speech bubbles, panels, avatars
- **Font Awesome Icons**: Visual expression through icons
- **Animation Effects**: CSS animations like bounce, sparkle

## Character Examples

- **Questioner**: Raises questions on behalf of readers
- **Explainer**: Provides expert knowledge clearly
- **Third-party View**: Offers objective opinions
- **Practitioner**: Shares practical experiences

## Usage Example

**Input**: Technical article "New Python Features"
**Output**: Comic HTML with programmer and Python expert dialogue

## Input Requirements

- Text to convert
- Target audience (optional)
- Character specifications (optional)

## Output Format

Complete HTML file (CSS, Font Awesome included)
