# Deep Research Quick Explainer V1

## Intended Use

- Explain latest technologies and topics in an easy-to-understand way for non-experts
- Explain innovation and benefits of technology in simple terms
- Create explanations including how to use

## Prompt

```
Please explain ＯＯＯＯ in a way that even people without knowledge can understand what is innovative and what are the benefits, and also describe how to use it.
```

## How to Use

1. Enter the above prompt into AI chat tool
2. Replace ＯＯＯＯ with the technology or topic you want to explain
3. Explanation text will be generated

## Input Example

```
Please explain deep learning in a way that even people without knowledge can understand what is innovative and what are the benefits, and also describe how to use it.
```

## Output Example

- Overview and innovation of the technology
- Problems that can be solved by the technology
- Specific benefits and application examples
- Basic usage and getting started

## Notes

- Explain technical terms in simple language
- Use concrete examples for clarity
- Don't assume reader's prior knowledge
