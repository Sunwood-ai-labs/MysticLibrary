# Colormap Generator v1.0

A prompt for extracting beautiful color palettes from images and generating interactive colormap HTML.

## Use Cases

- Extract color palettes from images
- Generate interactive color map HTML
- Create color tools for design projects

## How to Use

1. Paste the prompt into an AI
2. Upload an image
3. Save and use the generated HTML

## Features

- Color analysis from images
- 6-8 primary colors extraction
- Color naming with creative names
- Interactive HTML output

## Notes

- Use Kaisei Decol font for all text
- Mobile-first responsive design
- Consider accessibility
