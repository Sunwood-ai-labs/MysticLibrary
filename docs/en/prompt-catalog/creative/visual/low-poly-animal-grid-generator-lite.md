# Low-Poly Animal Grid Generator Prompt V1 Lite

## Intended Use

- When you want to quickly generate animal icon sets
- When you prefer a simple, easy-to-use prompt
- When you value consistency in low-poly style
- When you need clean, vector-friendly designs

## Prompt Text

```text
Nine low-poly [ANIMAL] icons in a clean 3 × 3 grid, white background.
Each icon: triangular facets, flat solid colors only, bold black outline, retro palette (teal, coral, mustard, navy, ivory, white), tiny sparkle accent.
Vector-friendly, sharp edges, square overall.
```

## How to Use

1. Replace `[ANIMAL]` with any animal type (e.g., cat, dog, bird, fish)
2. Input to image generation AI
3. Adjust color variations as needed

## Input Example

```text
Nine low-poly bird icons in a clean 3 × 3 grid, white background.
Each icon: triangular facets, flat solid colors only, bold black outline, retro palette (teal, coral, mustard, navy, ivory, white), tiny sparkle accent.
Vector-friendly, sharp edges, square overall.
```

## Output Example

- Nine different bird icons arranged in a 3×3 grid in low-poly style
- Clean style composed of triangular facets
- Retro palette (teal, coral, mustard, navy, ivory, white)
- Bold black outlines and small sparkle accents
- Vector-friendly with sharp edges that remain crisp at any resolution

## Notes

- This is the lite version of V1. Use [V1 (detailed)](./low-poly-animal-grid-generator.md) for more detailed specifications
- Outlines are bold with high overall contrast
- Background is always white
- Simple one-line prompt makes customization easy
