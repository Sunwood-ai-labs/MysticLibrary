# Werewolf Simulator (NanJ Style)

## Intended Use

- When you want to simulate a werewolf game with anime characters
- When you want output in an entertaining thread format
- When you want to see character thinking patterns and reasoning

## Prompt

```text
Simulate a werewolf game with OOOO characters and creatively write it as a NanJ-style bulletin board thread.

## Format Requirements
- Write in 2channel/NanJ-style thread format
- Must include response number, name (character name), posting date/time, and ID
- Write anchors in full-width characters as ">>"
- Faithfully reproduce each character's tone and personality

## Game Settings
- **Participants**: 8-12 anime characters
- **Roles**: Villagers, Werewolves, Seer, Bodyguard, Medium, etc.
- **Phases**: Alternate between Day (discussion/voting) and Night (ability execution)
- **Victory Conditions**: Village faction wins by eliminating all werewolves; Werewolf faction wins by reducing villagers to equal or fewer numbers

## Performance Instructions
1. **Character Reflection**
   - Faithfully reproduce each character's tone, personality, and thinking patterns
   - Have them reason and suspect in character
   - Utilize relationships and settings from the original work

2. **Thread Atmosphere**
   - Mix in NanJ-style taunts and live commentary appropriately
   - Use NanJ quotes like "Grass" (lol), "Shiver...", "Capable"
   - Include consecutive posts and short responses during exciting moments

3. **Game Progression**
   - Clearly mark the beginning and end of each phase
   - Dramatically present voting results and ability execution results
   - Make eliminated players' last words memorable

4. **Discussion Quality**
   - Have each character use both logical thinking and emotional judgment
   - Balance serious reasoning segments with heartwarming moments
   - Create a compelling and persuasive narrative
```

## How to Use

1. Replace `OOOO` with the title of the character work
2. You can optionally specify additional werewolf game settings

## Input Example

```text
Simulate a werewolf game with Attack on Titan characters and creatively write it as a NanJ-style bulletin board thread.
```
