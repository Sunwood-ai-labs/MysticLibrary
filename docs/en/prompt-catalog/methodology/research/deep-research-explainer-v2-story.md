# Deep Research Explainer V2 (Story Style)

## Intended Use

- When creating explanatory content for younger audiences
- When you want to introduce technology in a narrative format
- When creating explanation articles prioritizing readability

## Prompt

```text
Explain what is innovative and beneficial about OOOO in a way that even people without knowledge can understand, and also include instructions on how to use it.

Also, write in a story style that high school students can enjoy reading.
```

## How to Use

1. Replace `OOOO` with the topic you want to explain
2. Request output in a story-style narrative

## Input Example

```text
Explain what is innovative and beneficial about Transformers in a way that even people without knowledge can understand, and also include instructions on how to use them.

Also, write in a story style that high school students can enjoy reading.
```

## Output Example

The narrative format will include:

- Introduction of technology through characters and settings
- Explanation of innovation
- Benefits and advantages
- How to use

## Notes

- Compared to V1, you get explanations in a more approachable style
- Consider the balance between technical accuracy and readability
