# Deep Research Explainer V3.1 (Light Novel with Kishoutenketsu)

## Intended Use

- When you want to introduce technology with a four-part story structure
- When creating self-contained explanatory content as a story
- When you want structural explanations in a light novel style

## Prompt

```text
Explain what is innovative and beneficial about OOOO in a way that even people without knowledge can understand, and also include instructions on how to use it.

Also, write in a light novel style that high school students can enjoy reading, with attention to the four-part structure (Ki-Sho-Ten-Ketsu).
```

## How to Use

1. Replace `OOOO` with the topic you want to explain
2. Request output in a light novel style with Ki-Sho-Ten-Ketsu structure

## Input Example

```text
Explain what is innovative and beneficial about generative AI in a way that even people without knowledge can understand, and also include instructions on how to use it.

Also, write in a light novel style that high school students can enjoy reading, with attention to the four-part structure (Ki-Sho-Ten-Ketsu).
```

## Output Example

The four-part structure will include:

- **Ki (Introduction)**: Introduction, problem statement (setting background and challenges)
- **Sho (Development)**: Development (technical explanation, innovation description)
- **Ten (Twist)**: Twist (surprise element, new perspective, specific applications)
- **Ketsu (Conclusion)**: Conclusion (summary, how to use, future outlook)

- This version adds the Ki-Sho-Ten-Ketsu structure to V3
