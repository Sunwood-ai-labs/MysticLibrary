# Part 1: Fundamentals of Rap

## 1. Basic Rhyme Techniques

### 1.1 Single Rhyme
- Definition: Rhyme on one syllable
- Characteristics: Beginner-friendly, basic technique

### 1.2 Multi Rhyme
- Definition: Rhyme on two or more syllables
- Characteristics: Rhythmic with depth

### 1.3 Internal Rhyme
- Definition: Rhyme within a line
- Characteristics: Adds flow variation

### 1.4 End Rhyme
- Definition: Rhyme at line end
- Characteristics: Most common technique

## 2. Advanced Techniques

### 2.1 Assonance and Consonance
- Assonance (vowel rhyme)
- Consonance (consonant rhyme)

### 2.2 Homophones
- Wordplay using same-sounding words

### 2.3 Chain Rhyme
- Rhythm construction with continuous rhymes

# Part 2: SUNO Style Guidelines

## 1. Structure Elements

### 1.1 Basic Structure
[Intro] (4-8 lines) - Theme introduction and world-building
[Verse 1] (8-12 lines) - Main story development
[Hook] (4-6 lines) - Impressive chorus
[Verse 2] (8-12 lines) - Theme deepening and development
[Bridge] (4-6 lines) - Transition point
[Outro] (4-6 lines) - Message conclusion

### 1.2 Section Characteristics
- Intro: Theme presentation and flow establishment
- Verse: Detailed storytelling
- Hook: Impressive phrase repetition
- Bridge: Introduction of new perspective
- Outro: Powerful conclusion

## 2. Style Characteristics

### 2.1 Musical Features
- Beat: 16-beat based
- Flow: Melodic development
- Rhythm: 2-4 bar groupings
- Characteristic: Fusion with rock elements

### 2.2 Expression Style
- Language: Mix of modern youth language and street slang
- Metaphor: Effective use of metaphor
- Message: Fusion of sociality and hope
- Tone: Powerful declarative expression

# Constraints

## 1. Expression Constraints
- Exclude excessive violence
- Prohibit discriminatory expressions
- Avoid direct social criticism
- Limit pessimistic messages

## 2. Technical Constraints
- Consistency of syllable count per line
- Natural placement of rhymes
- Flow consistency
- Smooth connections between sections
