# Amped-Up Enka Lyrics Generator

## Intended Use

- Create high-tempo high-energy enka-style lyrics
- Generate lyrics in SUNO format
- Generate Japanese enka-style lyrics

## Prompt

```text
# Amped-Up Enka Lyrics Generation Prompt

## Requirements
- Japanese lyrics
- High-tempo and high-energy
- Enka-style

## Music Generation AI "Suno" Tag Cheat Sheet

Tag Structure
[Intro] - Song introduction
[verse] - Regular verse
[Verse 2], [Verse 3] - Verse continuation
[chorus] - Repeating theme (chorus)
[pre-chorus] - Section leading to chorus
[post-chorus] - Section immediately after chorus
[hook] - Catchy memorable musical phrase
[bridge] - Section contrasting verse and chorus
[break] - Rest or instrumental break
[interlude] - Musical break between sections
[outro] - Song ending section
[solo] - Instrumental solo section
[instrumental] - Instrumental only section
[build-up] - Gradually increasing tension and intensity
[fade-out] - Gradually decreasing volume to end

Performance Tags
[powerful] - Powerful vocals
[forte] - Strong volume, forte
[belting] - Powerful and emotional
[intense] - Emotional dynamic intensity
[climax] - Most exciting part of the song
[passionate] - Deeply emotional vocal performance
[soaring] - Magnificent moving effect, dramatically raising voice to high register

## Lyrics Structure Example

[Lead In - Electric guitar and drums]
[Intro]
First verse lyrics

[Break]
[Interlude]
[Instrumental: Guitar]

[Verse]
Second verse lyrics

[Hook]
Catchy refrain

[Chorus]
Chorus section

[Verse]
Third verse lyrics

[Hook]
[Chorus]

[Bridge]
Bridge section

[Key Change]
[Hook]
[Chorus]

[Outro]
Ending

[End]
```

## How to Use

1. Determine theme and song tone
2. Enter prompt into AI
3. Lyrics generated in SUNO format

## Input Example

```
Theme: People dancing in the city at night
Tone: High-energy, upbeat
```

## Output Example

High-tempo high-energy enka-style Japanese lyrics generated in SUNO format.

## Notes

- Japanese lyrics are generated
- Includes melodic expressions characteristic of enka
- High-energy danceable tune
