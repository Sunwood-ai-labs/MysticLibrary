# Graphic Recording Thumbnail Prompt V2

## Overview
This prompt generates high-quality blog article thumbnails like those created by top-tier consultants. It creates visually polished, infographic-style thumbnails with 16:9 aspect ratio that convey content at a glance.
For the color palette, refer to the format below and create a modern, stylish color palette.

## Basic Configuration
```
Create a thumbnail for [article title] in HTML/CSS
Make it 16:9 size (1920x1080px)
Enhance infographic readability and professionalism by utilizing Font Awesome icons
Also use the color map and fonts specified below
```
## Color Palette Definition (Customizable)

Japanese modern feel

## Design Elements
Please include the following elements:
1. **Title Section**:
   - Main title (large, prominent font - use Kaisei Decol)
   - Subtitle (supplementary information - use Zen Kurenaido)
   - Relevant Font Awesome icons
2. **Visual Elements**:
   - Related graphics/illustrations on the right side
   - Badges and labels (indicating topics or categories)
3. **Content Features**:
   - Display 3-4 main points with icons
   - Keep each point short and concise
4. **Professional Elements**:
   - Gradient background
   - Subtle grid pattern
   - Appropriate shadow effects
