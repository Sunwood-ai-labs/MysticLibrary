# Instagram用画像スライドインフォグラフィック作成プロンプト V2.0

## 概要
Instagram投稿用の正方形(600×600px)画像スライドを**任意の枚数**作成し、個別または一括でダウンロードできるおしゃれなインフォグラフィックを生成するためのテンプレートです。

## 基本仕様

### 技術要件
- **フォーマット**: SVGベースのHTML
- **サイズ**: 600×600px(正方形)
- **フォント**: Kaisei Decal(メイン)
- **アイコン**: Font Awesome 6(多用)
- **出力**: 高品質PNG画像
