【実行手順と操作の厳守ルール】
1. 構築完了後、アプリ一覧(Your apps)に戻る操作はスキップしてください。
2. 編集画面右上の「GitHub」ボタンを押し、アプリをデプロイしてください。
3. 次に「Share app」をクリックします。URLが長いため、テキストを直接読み取るのではなく、必ず「Copy」ボタンをクリックしてクリップボード経由で完全なURLを取得し、ここに提示してください。
4. 公開とURL取得が完了したら、画面左下のチャット入力欄(Make changes, add new features...と書かれたボックス)を使用して、AI Studioのコードアシスタントに以下の追加指示を直接出してください。

追加指示内容:
「このアプリのhttps://t.co/lPlBAYWdkGを整備してください。非常に洗練されたプロフェッショナルなデザイン(Markdown装飾の活用)にし、日本語と英語の両方の言語で内容を記述してください。」

5. 最後に、生成されたREADMEの内容と共有URLを私に報告して完了してください。
